# 入職門檻與學歷規劃

```mermaid
flowchart LR
    BS["學士（大學部）"] --> BS_J["可入職：<br/>設備工程師<br/>測試工程師<br/>Layout（初級）<br/>製程工程師（生產線）<br/>IE / 廠務"]
    MS["碩士（研究所）"] --> MS_J["主流入職：<br/>IC Design<br/>Verification / DFT<br/>製程 R&D<br/>EDA / CAD<br/>可靠度工程師"]
```

## 各學歷的起點薪資差異（台積電為例）

| 學歷 | 入職職位 | 新鮮人年薪（TWD）|
|------|---------|-------------|
| 學士 | 工程師（生產 / 設備） | NT$700K – NT$900K |
| 碩士 | 工程師（R&D / 設計）| NT$900K – NT$1.4M |
| 博士 | 資深工程師 | NT$1.5M – NT$2.0M |

> 台積電有「直攻計畫」專為學士生設計，提供較快的升遷路徑
