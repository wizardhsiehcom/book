# 工業工程師（IE）

工業工程師（Industrial Engineer / IE）在晶圓廠負責最佳化製造流程，讓晶圓生產更快、更有效率、更低成本。他們不碰製程配方，而是從系統層次分析整個工廠的運作。

## 核心工作

**每天在做什麼：**
- **產能規劃（Capacity Planning）**：建立晶圓廠產能模型，平衡各製程模組的設備負載
- **瓶頸分析**：找出製程流程中的瓶頸（等待最長的設備）；用排隊理論分析
- **週期時間縮短（Cycle Time Reduction）**：減少在製品（WIP）庫存；導入拉式生產（Pull System）
- **精實生產 / 六標準差**：推動 Kaizen 改善活動；DMAIC 方法論
- 與 MES（製造執行系統）協作最佳化 Lot 派工邏輯

## 與製程 / 設備工程師的差異

| | IE 工業工程師 | Process Engineer | Equipment Engineer |
|-|-------------|-----------------|-------------------|
| 關注層次 | 工廠系統層次 | 單一製程步驟 | 單一設備機台 |
| 主要工具 | 模擬軟體、SQL、Python | SPC、DOE | PM 程序、維修 |
| 無塵室工作 | 少（主要在辦公室）| 多（每天進出）| 大量 |
| 技術背景 | 工業工程 / OR | EE / 物理 / 材料 | ME / EE / 化工 |

## 核心技能

- 工業工程 / 作業研究（Operations Research）背景
- 工廠模擬工具（Arena、Factory Works）
- SPC、DOE、迴歸分析
- Python / SQL（大量數據分析）
- 精實六標準差認證（Green Belt 或以上）加分

## 主要雇主

台積電（IE 部門規模大）、聯電、日月光、所有晶圓廠 / OSAT 封測廠
