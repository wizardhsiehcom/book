## 1. ch13 課程總結表把「SMC」列入「形式保證」——**有誤**

**書中現況**（`13-runtime-monitoring.md` 總結表）：「形式保證｜可達性分析、SMC｜量化或証明失效機率的上界」。

**結論：歸類錯誤。** 在本課程／教科書語境中，SMC 專指 **Sequential Monte Carlo（循序蒙地卡羅）**：教科書目錄中 Sequential Monte Carlo 是第 7 章 **Failure Probability Estimation（失效機率估計）** 的一節（§7.4，pp. 177–181），與 Direct Estimation、Importance Sampling、Adaptive Importance Sampling、Multilevel Splitting 並列，屬**失效分析（估計失效機率）**範疇，是不含形式保證的抽樣估計法。本書 ch12 自己也把 SMC 介紹為失效機率估計演算法（12.5 節），ch13 的歸類與 ch12 自相矛盾。

「Statistical model checking」（另一個縮寫也是 SMC、且確實常被歸入帶機率保證的驗證方法）並未在 AA228V 課程或教科書中出現（教科書索引無此詞條），不能作為此歸類的辯護。

**正確說法**：形式保證一列應為「可達性分析（含機率可達性，見教科書第 10 章 Probabilistic Reachability 一節）」；SMC 應移入「失效分析」一列（該列現有「蒙地卡羅、最佳化偽證」，可補「重要性採樣、SMC」），或乾脆刪去。另「証明」為異體字，應作「證明」。

依據：教科書目錄與 §7.4（pp. 177–181），2026-07-10 驗證。

## 2. Shapley 計算量：ch14 合理、ch15 口徑錯置——**ch15 有誤，需統一口徑**

**數字事實**：
- 2^24 = 16,777,216（約 1,677 萬）；2^25 = 33,554,432（約 3,355 萬）
- 2^40 ≈ 1.10 × 10^12（約一兆）
- 40! ≈ 8.16 × 10^47

**ch14**（「25 個格子就有超過 1600 萬種子集」）：**合理**。Shapley 值定義（教科書 eq. 11.2，p. 302）對特徵 i 的求和範圍是 I∖{i} 的所有子集，共 2^(n−1) = 2^24 ≈ 1,677 萬個，「超過 1600 萬」正確。若指全部特徵的全部子集則是 2^25 ≈ 3,355 萬，建議把語句寫明口徑：「對單一格子求 Shapley 值需對其餘 24 格的 2^24 ≈ 1,677 萬個子集求和」。

**ch15**（「計算複雜度為 O(2^n)，40 個時間步即需約 10^47 次計算」）：**有誤（口徑錯置）**。O(2^n) 是**子集口徑**，n = 40 時 2^40 ≈ 1.1 × 10^12，離 10^47 差了 35 個數量級。10^47 對應的是**排列口徑**：教科書 eq. 11.3（p. 303）把 Shapley 值改寫為對所有 n! 個排列求平均，40! ≈ 8.16 × 10^47。兩個口徑在課程中都出現（子集定義 eq. 11.2、排列改寫 eq. 11.3，後者才是蒙地卡羅近似演算法 11.4 的基礎），但不能把 O(2^n) 和 10^47 湊在同一句。

**建議統一說法**（二擇一）：
1. 「以子集定義計算為 O(2^n)：40 個時間步約需 10^12 次評估」；或
2. 「若以排列形式定義（對 n! 個排列平均），40 個時間步約有 8 × 10^47 種排列」。

若希望 ch14/ch15 呼應，建議都採子集口徑並各自標明 2^24 ≈ 1,677 萬、2^40 ≈ 10^12，另加一句「排列形式則為 n!（40 步約 8×10^47），實務上皆以蒙地卡羅抽樣近似」。

依據：教科書 §11.3.2（pp. 302–304），2026-07-10 驗證。

## 3. ch16 HJI PDE、安全控制器、BRT 定義——**公式正確；BRT 文字定義措辭有誤**

**(a) HJI PDE「∂V/∂t + min(0, H(x,∇V)) = 0」：正確（標準形式之一）。** 這是 Mitchell–Bayen–Tomlin (2005) 對 backward reachable tube 的時間相依 HJI 提法（在 Hamiltonian 外取 min{0,·} 以「凍結」已進入目標集的軌跡，對應 payoff 取時間最小值 J = min_t L(x(t))），與 ch16 前文的累積安全獎勵定義自洽。文獻中另一等價常見寫法是變分不等式（HJB/HJI-VI）：min{∂V/∂t + H, L(x) − V(x,t)} = 0, V(x,T) = L(x)——Bansal 實驗室自己的論文（本講 TaxiNet 案例的原始論文，arXiv:2309.13475 eq. 6）用的就是 VI 形式，且因控制器已固定為 g(x)，Hamiltonian 直接為 H = ⟨∇V, f(x, g(x))⟩、無需對 u 最佳化——這與 ch16「求解簡化 HJI PDE（代入 g(x)，無需最佳化控制）」的敘述一致。結論：書中公式可保留，建議加註「另一常見等價寫法為變分不等式 min{∂V/∂t + H, L − V} = 0」。

**(b) 安全控制器 u\* = argmax_u ∇V·f(x,u)：正確（無擾動情形）。** 與 Bansal et al. 2017 綜述 eq. (14)（a\* = argmax_a min_b λ·f(x,a,b)）一致；書中省略了擾動，若有擾動 d 應為 u\* = argmax_u min_d ∇V·f(x,u,d)。ch16 上文的價值函數 V = max_u min_d J 有寫出擾動，控制器公式卻無，建議補一句「有擾動時對 d 取 min」。另實務上此控制器通常只在 V ≈ 0 的邊界上以最小干預（least-restrictive）方式啟用，可視需要補充。

**(c) BRT 文字定義「無論系統採取何種最佳控制行動，軌跡終將進入失效集合」：措辭有誤（自相矛盾）。** 「無論……何種**最佳**控制」語意不通：最佳控制只有一個，「無論何種」應搭配「控制」而非「最佳控制」。標準定義（Bansal et al. 2017 §IV：G(t) 是 "the set of states from which the disturbance can drive the system to the actual unsafe set **despite the best control efforts**"）應譯為：「**即使採取最佳的（極力避開失效的）控制行動，軌跡仍無法避免在時間範圍內進入失效集合**的所有初始狀態集合」，或等價地「對**所有**控制策略，軌跡終將進入失效集合」。另注意：在本講的壓力測試設定中控制器已固定為 g(x)，此時 BRT 就是「閉迴路系統會進入失效集合的初始狀態集合」（arXiv:2309.13475 eq. 4），沒有「最佳控制」可言——ch16 若保留通用定義，宜在案例處說明此簡化。

依據：Bansal et al. 2017 綜述 §III–V（[arXiv:1709.07523](https://arxiv.org/abs/1709.07523)）、Mitchell et al. 2005 Thm. 2、[arXiv:2309.13475](https://arxiv.org/abs/2309.13475) eqs. (4)–(7)，2026-07-10 驗證。

## 4. ch10 Metropolis-Hastings：文字（對稱核）與代碼（非對稱修正）——**自洽且正確**

- 文字部分：「當核函數對稱（g(τ|τ′) = g(τ′|τ)，如高斯核）時 α = min(1, p̄(τ′)/p̄(τ))」——正確，這正是 Metropolis（1953）特例；教科書 p. 144 同樣寫明 "When the kernel is symmetric … the acceptance criteria reduces to p̄(τ′)/p̄(τ)"。
- Julia 代碼：`rand() < (p̄(τ′) * pdf(g(τ′), τ)) / (p̄(τ) * pdf(g(τ), τ′))`——這是**一般（非對稱）MH 接受率** p̄(τ′)g(τ|τ′) / (p̄(τ)g(τ′|τ))（`g(τ′)` 是以 τ′ 為中心的提案分佈，`pdf(g(τ′), τ)` = g(τ|τ′)），與教科書 eq. (6.3) 及演算法 6.2（pp. 144–145）**逐字元一致**。省略 min(1,·) 不影響正確性（比值 >1 時 rand()<比值 必然成立）。
- 自洽性：代碼是一般式，文字說的是對稱特例（此時修正項比值為 1，退化為 Metropolis），兩者相容。書中把演算法虛擬碼標題寫成「對稱核函數版本」而 Julia 代碼是一般版本，建議加一句過渡（「以下 Julia 實作為一般非對稱版本」）以免讀者誤會，但**無技術錯誤**。

依據：教科書 §6.3.1（pp. 144–146），2026-07-10 驗證。

## 5. ch12 SMC 權重更新與 p̂_fail = (1/M)Σw——**與教科書一致，正確**

教科書 eq. (7.21)（p. 178）：w_i^(ℓ) = w_i^(ℓ−1) · ḡ_{ℓ+1}(τ_i^(ℓ)) / ḡ_ℓ(τ_i^(ℓ))，其中 ḡ_ℓ 為未正規化中間密度、ḡ_1 = p（標稱分佈）；eq. (7.22)（p. 179）：p̂_fail = (1/m) Σ_i w_i^(n−1)，即**最終權重的平均**。書中 ch12 的寫法 w^(k) = w^(k−1) · G_k(τ^(k−1))/G_{k−1}(τ^(k−1))、p̂_fail = (1/M)Σ w^(N) 與此語意相同——都是「下一個分佈密度 ÷ 樣本當前所代表分佈的密度，在該樣本上取值」，只是索引記法不同（教科書在樣本已 MCMC 轉移到 ḡ_ℓ 後乘上 ḡ_{ℓ+1}/ḡ_ℓ；書中在轉移前乘上 G_k/G_{k−1}，兩者為同一遞推的平移標號）。初始權重 w^(1) = 1 與教科書代碼 `ws = [ḡs[1](τ)/p(τ)]` 在 G_1 = p 時一致。書中亦正確指出此結果的理論根據是正規化常數之比（教科書 §7.5，Llorente et al. 2023）。

兩點可補充（非錯誤）：(1) 教科書演算法 7.7 在每次迭代含**重採樣**步驟，且重採樣後須把權重重設為重採樣前的平均值，估計式 (7.22) 才保持正確——ch12 若提到重採樣，建議一併說明權重重設；(2) 各 G_k 必須是未正規化密度、末端 G_N ∝ p·1[fail]，其正規化常數即 p_fail。

依據：教科書 §7.4–7.5（pp. 177–181）、演算法 7.7（p. 181），2026-07-10 驗證。

## 6. ch4 STL 強健性各運算子——**正確**

教科書 eqs. (3.11)–(3.16)（p. 79，引 Donzé & Maler 2010 的量化語意）：

- ρ(¬P) = −ρ(P) ✔（取負）
- ρ(P∧Q) = min(ρ(P), ρ(Q)) ✔
- ρ(P∨Q) = max(ρ(P), ρ(Q)) ✔
- ρ(□_[a,b] P) = min_{t′∈[t+a,t+b]} ρ(P) ✔（時間軸取 min）
- ρ(◇_[a,b] P) = max_{t′∈[t+a,t+b]} ρ(P) ✔（時間軸取 max）
- 基本謂詞 s_t > c 的強健性 s_t − c ✔

ch4 所列與教科書完全一致，無誤。ch4 未列 Until（教科書 eq. 3.17）與蘊含（ρ(P→Q) = max(−ρ(P), ρ(Q))），屬取捨、非錯誤。平滑強健性（soft-min/soft-max、參數 w→∞ 趨近真極值）的敘述亦與教科書一致。

依據：教科書 §3.5（pp. 77–79），2026-07-10 驗證。

## 7. ch8 幾何分佈期望 1/p_fail、ch11/12 最佳 IS 提案零變異數——**均正確**

**(a) 幾何分佈**：首次成功（找到失效）所需試驗次數 K ~ Geometric(p_fail)，P(k) = (1−p_fail)^(k−1) p_fail，E[K] = 1/p_fail。教科書 eq. (4.1) 與隨文說明（p. 94）：「The expected value of this distribution, 1/p_fail, corresponds to the average number of samples required to find a failure」，並同樣以 10^−9 失效率的航空系統（平均需十億次模擬）為例。ch8 敘述正確。

**(b) 最佳 IS 提案**：教科書 eq. (7.14) 給出 IS 估計器變異數 Var[p̂_fail] = (1/m) E_q[(p(τ)1{fail} − q(τ)p_fail)²/q(τ)]，eq. (7.15)（p. 167）：取 q\*(τ) = p(τ)1{τ 為失效}/p_fail（即失效分佈 p(τ|fail)）時變異數為**零**；且因正規化常數 p_fail 正是待估量而形成循環困境。ch11 與 ch12 的公式、零變異數論述、循環困境說明皆與教科書一致，正確。ch12 補充的覆蓋條件（q\*(τ)>0 ⇒ q(τ)>0，否則估計有偏）亦為標準要求，正確。

依據：教科書 §4.1（p. 94）、§7.2.1（p. 167），2026-07-10 驗證。
