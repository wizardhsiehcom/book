# Stanford AA228V: Validation of Safety Critical Systems - 繁體中文書籍轉換計畫

## 核心目標
將史丹佛大學 AA228V 課程（安全關鍵系統的驗證）的完整影片逐字稿，結合官方 Julia 筆記資源，轉化為結構嚴謹、高可讀性的繁體中文 Markdown 教科書。

## 執行原則
與之前的專案相同：
1. **完整閱讀**：Agent 必須從頭到尾閱讀整份逐字稿。
2. **筆記與成書分離**：先產出高度抽象的個人筆記，再寫出流暢的書籍草稿。
3. **無縫接軌官方講義**：與 GitHub 的 Julia Notebooks 進行整合（如 `discrete_reach.jl`, `failure_analysis.jl` 等）。

## 階段劃分
- **Phase 1 (Batch 0)**: 導讀與基礎設定
- **Phase 2 (Batch 1-X)**: 平行處理影片逐字稿與對應的 Julia Notebook
- **Phase 3**: 統整與排版
