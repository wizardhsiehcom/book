# 第二十二章：注意力（續）與觸覺

## 1. 導讀

本章是注意力主題的下半部，同時開啟了全新感官——觸覺（Touch）的探索。在前半部，我們將目光從行為學層次轉向神經機制，探討當我們「注意」某個事物時，大腦中的單一神經元如何改變其活動模式；我們也會探討注意力與眼球運動（Eye movements）系統之間意想不到的緊密關聯，以及當頂葉受損時，為何會產生奇特的「半側忽視症」（Neglect）。

後半部將進入本課程探討的第三種感官：觸覺。與視覺、聽覺相比，觸覺（以及味覺、嗅覺）的研究相對較少，主因是觸覺刺激在實驗室中難以精確操弄。然而，觸覺是我們日常抓握、判斷材質、感受溫度的重要依據。讀完本章，你將了解皮膚下的四種關鍵機械性受器（Mechanoreceptors）如何分工合作，以及我們的神經系統如何巧妙結合空間與時間資訊，讓我們能精準感知從粗糙砂紙到滑順絲綢等各種紋理；此外，你也會認識痛覺與本體感覺的基礎，並了解大腦皮質中奇特的「觸覺小人」（Homunculus）地圖。

---

## 2. 核心概念：注意力的神經生理基礎

### 2.1 單一神經元層級的注意力調節
大腦造影（如 fMRI）雖然能看出腦區的整體活化，但要了解注意力的精確機制，必須觀察單一神經元的反應。在 Bob Desimone 等人的經典獼猴實驗中，研究人員記錄了視覺區域（如 V4）神經元的活動。
實驗中，神經元的「感受野」（Receptive field）內同時出現兩個刺激：一個能引起該神經元強烈反應（有效刺激），另一個則不會（無效刺激）。結果發現，當猴子將注意力集中在「有效刺激」時，神經元的放電頻率大幅增加；若注意力轉移到「無效刺激」時，放電頻率則降低。
**關鍵發現**是：這種注意力調節效應在兩個刺激**都在同一個感受野內**時最為顯著。換言之，注意力彷彿讓神經元的感受野「縮小」（Shrink）並緊緊包覆住被注意的目標物，藉此排除鄰近干擾。這種現象在 V4 區及高階的下顳葉（TE 區）很明顯，但在初級視覺皮質（V1）則幾乎觀察不到。

### 2.2 注意力與眼動系統的潛在關聯
通常我們注意某個地方時，眼睛也會看向那裡。大腦控制眼動的區域之一是「額葉眼動區」（Frontal Eye Fields, FEF）。如果用電極刺激 FEF 某個特定位置，會強制引發眼球運動（Saccade）到對應的視野位置。
Tierney Moore 的實驗提出了一個有趣的問題：注意力機制會不會與眼動機制是共用的？他們找到一對神經元——一個在 FEF（控制某個特定空間的眼動），另一個在 V4（負責感知該特定空間）。
當研究人員在 FEF 給予**閾值下**的微弱電刺激（不足以引發眼球轉動）時，竟然觀察到 V4 區域神經元對該空間的反應增強了！這與動物「主動注意」該空間時產生的神經反應增強如出一轍。這暗示了注意力的演化可能源自於眼動前置的運動計畫（Motor command）迴路。

## 3. 機制與現象：頂葉與半側忽視症

### 3.1 半側忽視症（Neglect）
頂葉（Parietal lobe，背側視覺路徑的終點）在注意力控制中扮演關鍵角色。如果頂葉（尤其是右側頂葉）因為中風而受損，患者經常會罹患「半側忽視症」。
這種患者並非視力受損，而是失去了對**左側空間**的「注意」能力。他們可能只吃盤子右半邊的食物、只畫時鐘右半邊的數字，甚至只穿戴右半邊的衣物。
這顯示注意力並非只是初階視覺的附屬品，而是一個可以獨立受損的高階認知功能。更有趣的是，這種忽視往往是「基於物件」（Object-based）的：即便一個物品完全位在患者未受損的右側視野，患者仍可能忽視該「物品本身」的左半側，這進一步支持了注意力經常是以「物件」為單位的理論。

## 4. 核心概念：觸覺與皮膚機械性受器

進入觸覺領域，多數的感知機制發生在週邊的「受器」層級。皮膚內含有多種機械性受器（Mechanoreceptors），它們依據兩個主要維度來分類：**適應速率**（快/慢）與**感受野大小**（小/大）。這構成了四種主要的受器（以猴子與人類模型為主）：

1. **Meissner corpuscle (RA / FA1)**：
   - 快速適應（Rapidly adapting），感受野小。
   - 位於淺層，對低頻振動與皮膚形變（如東西滑過手指）敏感。
2. **Merkel disc (SA1)**：
   - 慢速適應（Slowly adapting），感受野小。
   - 位於淺層，對持續的壓力與精細的空間結構敏感。
3. **Pacinian corpuscle (PC / FA2)**：
   - 快速適應，感受野大。
   - 位於深層，對高頻振動極度敏感。
4. **Ruffini ending (SA2)**：（註：猴子缺乏此受器）
   - 慢速適應，感受野大。
   - 對皮膚的拉扯與張力敏感。

### 4.1 受器敏感度與頻率包絡線
若用不同頻率的微小探針震動皮膚，並測量人類能感受到的最小振幅閾值，我們會得到一條觸覺的「頻率敏感度曲線」。這條曲線是由 SA1、FA1、FA2 各自的最佳敏感範圍交疊而成的，這與視覺中的對比敏感度函數（由不同空間頻率通道組成）以及暗適應曲線（由視錐、視桿細胞交接）有著異曲同工之妙。

## 5. 心理物理與證據：觸覺的空間與時間解析度

### 5.1 兩點閾值與空間解析度
觸覺受器在全身的分佈極度不均勻。透過「兩點閾值」（Two-point discrimination）測試（以兩根針同時戳刺皮膚，測量能分辨是「兩點」而非「一點」的最短距離），可以發現指尖與嘴唇的解析度極高（約數毫米內即可分辨），而手臂、背部的解析度極低（需相距數公分才能分辨）。這反映了小感受野受器（SA1, RA）在指尖的高密度分佈。

### 5.2 紋理（Texture）的時空編碼機制
當我們撫摸材質時，如何分辨粗糙的砂紙與平滑的絲綢？
神經生理實驗顯示，當帶有凸起字母或粗大點陣的滾輪掃過指尖時，**小感受野受器（SA1 與 RA）**的放電模式能完美地重現刺激的「空間圖像」。
然而，當面對如絲綢、尼龍等極其細緻的紋理時，空間解析度已無法區分它們。此時，觸覺系統轉而依賴**時間資訊**：手指滑過細緻紋理時會產生高頻的皮膚微小震盪。這會導致快速適應受器產生與震盪頻率「相位鎖定」（Phase-locking）的脈衝發射（類似聽覺神經對高達 400Hz 聲音的相位鎖定）。透過分析神經放電的頻譜（Spectrogram），大腦就能區分不同的細緻紋理。

## 6. 其他觸覺次系統與大腦皮質表徵

### 6.1 痛覺、溫度與本體感覺
- **本體感覺（Proprioception / Kinesthetic receptors）**：位於肌肉紡錘、肌腱與關節中，提供肢體位置的訊息。著名的病患 Ian Waterman 失去此神經後，必須完全依賴視覺來控制肢體動作。
- **溫度受器（Thermoreceptors）**：分為冷受器與熱受器，負責感測相對皮膚基準溫度（30-36°C）的變化。
- **痛覺受器（Nociceptors）**：
  - **A-delta 纖維**：具有髓鞘（Myelinated），傳導速度快。負責踩到樂高時第一瞬間那種快速、尖銳的痛覺。
  - **C 纖維**：無髓鞘（Unmyelinated），傳導速度慢。負責隨後那種緩慢、持續的跳動性疼痛，用以提醒身體持續保護患部。

### 6.2 體感覺皮質（S1）與觸覺搜尋
從受器傳入的資訊最終送達大腦的初級體感覺皮質（S1）。這裡存在著類似視覺「視網膜對應圖」的**「體位圖」（Somatotopic map 或 Homunculus）**。與視覺的皮質放大效應相似，大腦分配給指尖、臉部的皮質面積遠大於背部或手臂，反映了該部位受器密度的多寡。
在行為上，觸覺也展現出類似視覺搜尋（Visual search）的特徵。在雙手多指的「觸覺搜尋」任務中，尋找「粗糙之於平滑」、「硬之於軟」會產生「突顯效應」（Pop-out），反應時間不隨手指數量增加而變長；但與視覺不同的是，觸覺中的「特定方向（如水平之於垂直）」無法產生突顯效應。

## 7. 常見誤解與特別釐清
- **觸覺敏銳度與肌肉控制的關係**：雖然指尖與臉部有極高的觸覺解析度與精細的肌肉控制，但決定觸覺解析度的主因是皮膚層次「機械性受器」的密度，而非直接由運動神經元決定，只是兩者在演化上往往為了達成精細操作而共同發展。
- **神經元對方向的調諧（Orientation tuning）並非視覺專有**：在體感覺皮質中，使用針筒陣列壓迫皮膚，也能測出神經元對特定「方向」線條的偏好放電，展現出與視覺皮質相似的特徵萃取機制。

## 8. 小結
- 單一神經元實驗顯示，注意力會使視覺感受野「縮小」至目標物上。
- 額葉眼動區（FEF）的次閾值刺激能誘發視覺皮質的注意力增強反應，顯示注意力與眼動準備網路的高度重疊。
- 頂葉受損會造成半側忽視症（Neglect），患者會忽略一側（通常為左側）空間或物體的特徵。
- 皮膚主要包含四種機械性受器（Meissner, Merkel, Pacinian, Ruffini），根據適應速度快/慢與感受野大/小進行分類。
- 觸覺紋理的感知結合了低頻空間結構的影像對應，以及高頻微小震盪造成的相位鎖定頻譜特徵。
- 痛覺分為有髓鞘（快速尖銳）與無髓鞘（慢速持續）兩種纖維傳遞；體感覺皮質（S1）則具有依據受器密度放大的「觸覺小人」地圖。
- 點字（Braille）利用點狀突起而非真實字母形狀，是因為點陣在觸覺受器的低空間解析度下，具備更高的鑑別度。
