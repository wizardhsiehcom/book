# 第二十章：注意力 (Attention)

## 1. 導讀

在日常生活中，「注意力（Attention）」是一個我們非常熟悉的詞彙；但在知覺心理學中，注意力扮演著決定我們「看見」或「聽見」什麼的關鍵角色。本章將探討注意力在知覺處理中的運作機制。讀完本章後，讀者將了解注意力如何像一盞聚光燈般提升資訊處理的效率，認識自動引發與主動控制的兩種注意力路徑，並學習如何透過空間提示、多重物件追蹤、以及視覺與聽覺搜尋等經典實驗典範，來測量與量化注意力的極限與特徵整合的過程。

## 2. 核心概念

### 2.1 空間提示與聚光燈隱喻
在視覺中，我們通常會注意我們眼睛所看的地方（注視點），但注意力並不完全等同於眼球運動。你可以在眼睛直視前方的同時，將注意力轉移到視野周邊的物體上。為了研究這種空間注意力的轉移，認知心理學家 Posner 發展了著名的「空間提示典範（Cueing paradigm）」。

在實驗中，受試者需要盡快辨識出現在螢幕上的目標。在目標出現前，會先給予一個「提示（Cue）」，指出目標可能出現的位置。結果顯示，當受試者事先知道目標的位置時，他們的反應時間會顯著縮短。這種效益具有空間局部性：當提示位置與目標位置完全重合時，反應最快；隨著目標偏離提示位置，反應時間的改善幅度就會下降。這引出了一個經典的隱喻——**注意力就像一盞聚光燈（Spotlight metaphor）**：無論聚光燈照向哪裡，該區域內的資訊處理就會變得更快、更好。

### 2.2 兩種注意力的控制：外源性與內源性
提示如何引導注意力？研究發現有兩種截然不同的機制：
- **外源性提示（Exogenous cues）**：由外部刺激（如突然的閃光或移動）自動且反射性地吸引注意力。這是一種「由下而上（Bottom-up）」的控制。
- **內源性提示（Endogenous cues）**：由符號（如一個指向右邊的箭頭）引導，受試者必須先解讀符號的意義，然後主動將注意力轉移到目標位置。這是一種「由上而下（Top-down）」的控制。

這兩者的關鍵差異在於時間歷程。如果在提示與目標出現之間設定不同的時間差（Stimulus Onset Asynchrony, SOA），外源性提示大約只需 100 毫秒就能達到最大的注意力效益；而內源性提示因為需要高階的認知解讀與主動控制，則需要更長的時間才能產生明顯的加速效果。在現實生活中，這兩種機制經常交織在一起，例如看棒球時，你根據對比賽的理解預測球的位置（內源性），同時又被球棒擊球的聲音瞬間吸引（外源性）。

### 2.3 超越單一聚光燈：多重物件追蹤
注意力是否只能是一盞單一的聚光燈？Zhenon Pylyshyn 提出的「多重物件追蹤（Multiple Object Tracking, MOT）」實驗挑戰了這個觀點。在這個任務中，受試者會先看到螢幕上有一群完全相同的物體，其中幾個會短暫閃爍作為提示。接著所有物體開始在螢幕上隨機移動與交錯。最後，當移動停止時，受試者必須指出哪些是最初被提示的目標。

實驗結果顯示，人類有能力同時追蹤 4 到 5 個獨立移動的目標。雖然錯誤率會隨著追蹤數量的增加而上升，但這證明了注意力資源可以被分散或同時分配到多個空間位置。

### 2.4 聽覺注意力與串流追蹤
注意力同樣在聽覺中扮演關鍵角色，尤其是在解決「雞尾酒會問題」時。當兩個聲音（例如兩位女性的語音）同時出現時，它們的基頻（F0）與共振峰特徵會在空間中交織纏繞。要在這之中聽懂其中一個人的話，大腦必須在特徵空間中連續「追蹤」那個聲音的軌跡。

實驗證明，如果在其中一個聲音中加入微小的頻率變化（如抖音 Vibrato），擅長追蹤指定目標音的受試者（通常受過音樂訓練），能比不擅長的人更敏銳地偵測到目標音上的抖音。這表示選擇性注意力不僅能隔離目標，還能提高我們對目標特徵細節的敏感度。

## 3. 機制與現象：特徵整合與視覺搜尋

注意力的一大功能是將分散的特徵「綁定」在一起。在視覺搜尋（Visual Search）任務中，這點表現得特別明顯：
- **特徵搜尋與跳出效應（Feature Search & Pop-out）**：當你要在一群黃色圓點中找出一個藍色圓點時，目標會瞬間「跳出（Pop-out）」。這種搜尋依賴單一維度的特徵差異（如顏色、方向、大小），注意力會被自動吸引到該位置。
- **交集搜尋（Conjunction Search）**：如果目標沒有單一獨特的特徵，而是特徵的組合（例如在「藍色水平線」與「紅色垂直線」中尋找「藍色垂直線」），搜尋就會變得很緩慢。這時，受試者必須像使用聚光燈一樣，逐一檢查（Serial search）畫面上的元素。

### 3.1 錯覺結合（Illusory Conjunctions）
心理學家 Anne Treisman 根據視覺搜尋的結果提出了「特徵整合論（Feature Integration Theory）」。她主張，大腦早期視覺系統會平行處理各種單一特徵（如顏色、形狀），形成不同的「特徵地圖」。但要將「紅色」和「垂直」正確地組合在同一個物體上，就必須依賴注意力作為黏著劑。

這項理論的一個強烈證據是**錯覺結合**現象。如果在螢幕上快速閃過一些彩色的字母，並要求受試者將注意力集中在螢幕中央讀取數字，他們在回報周邊字母時，經常會將特徵錯誤地組合起來（例如看到藍色的 T 和紅色的 X，卻報告說看到了紅色的 T）。這證明了當缺乏注意力時，特徵會處於游離狀態，無法被正確綁定。

## 4. 心理物理與證據

視覺搜尋實驗的核心測量方式，是分析**反應時間（Reaction Time）**與**集合大小（Set Size，即畫面中干擾物的數量）**的關係：
- 在發生跳出效應的**特徵搜尋**中，斜率為零。也就是說，不管畫面上有 5 個還是 50 個干擾物，找到目標的時間幾乎一樣快。
- 在需要集中注意力的**交集搜尋**中，反應時間會隨集合大小線性增加。值得注意的是，當「目標缺席（Target absent）」時，搜尋斜率通常是「目標存在（Target present）」的兩倍。因為如果目標存在，平均只要搜尋一半的物品就會找到；但如果目標不存在，受試者必須檢查完每一個物品（exhaustive search）才能確定。

## 5. 常見誤解

- **注意力等同於視線？** 錯誤。雖然我們通常會注意我們看的地方，但空間提示實驗證明了「隱性注意力（Covert attention）」的存在，即眼睛不動也能轉移注意力。
- **外源性與內源性是完全互斥的？** 在真實世界中，這兩者往往是協同運作的。我們通常同時擁有對情境的內在理解（內源性），並隨時準備對突發的環境變化（外源性）做出反應。
- **特徵整合論是絕對的真理？** 雖然錯覺結合現象很經典，但 Treisman 的理論在後來受到了挑戰。例如，某些複雜的高階特徵（如 3D 形狀）也能產生跳出效應，這顯示大腦並非只有最基本的特徵地圖能在注意力介入前被處理。

## 6. 小結

- 注意力可以決定我們在複雜環境中知覺到哪些資訊，並提升該資訊的處理效率。
- 注意力可比喻為一盞聚光燈，可由外部刺激自動吸引（外源性），或由內在意圖主動引導（內源性）。
- 多重物件追蹤實驗顯示，注意力可以同時分配給多個獨立移動的目標。
- 在聽覺中，注意力能幫助我們在相似的聲音軌跡中鎖定單一來源，並提升對其特徵的敏感度。
- 視覺搜尋實驗證明，單一特徵差異能產生平行的「跳出效應」，而特徵交集則需要耗時的序列搜尋。
- 當注意力資源不足時，大腦可能會將不同物體的特徵錯誤地結合在一起，產生錯覺結合。
