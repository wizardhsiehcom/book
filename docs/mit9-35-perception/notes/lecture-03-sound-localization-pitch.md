## 本講主問題
本講旨在收尾前一講的聲音定位（Sound Localization），特別補足「距離知覺（Distance Perception）」的機制。接著插入一段重要的方法學教學——心理物理學方法（Psychophysical Methods），解釋如何科學地測量人類感官的「閾值（Threshold）」，並介紹訊號偵測理論（Signal Detection Theory）以區分參與者的感知敏感度與反應偏好。原本預計討論的音高、音色與響度因時間關係被延至下一講。

## 核心概念
| 概念 | 解釋 | 處理方式 |
|---|---|---|
| 距離知覺 (Distance perception) | 透過直達聲與殘響的比例（Direct-to-reverberant ratio）以及對熟悉音源的預期音量來判斷聲音遠近。 | 於「聲音定位」段落收尾處詳細說明。 |
| 閾值 (Threshold) | 分為絕對閾值（最小可聽見的訊號，如聽力圖）與差異閾值（最小可辨識的變化，即 JND）。 | 於心理物理學段落開頭定義。 |
| 心理測量函數 (Psychometric function) | 描述受試者表現（如正確率）隨刺激強度變化的 S 型曲線，並非階躍函數。閾值通常定義於某個任意的表現標準（如 75%）。 | 搭配作圖概念解釋其在閾值測量中的意義。 |
| 訊號偵測理論 (SDT) | 用以解決「單一區間是/非任務」中的偏差問題。透過擊中率(Hit)與假警報率(FA)計算 d' (敏感度)，分離判斷標準 (Criterion)。 | 獨立子節重點講解 d' 與 Criterion 的觀念。 |
| 測量程序 (Measurement procedures) | 常數刺激法（預設多個刺激強度，得完整曲線但效率低）與適應性階梯法（根據前次答對與否調整難度，有效率找到閾值）。 | 以表格或並列方式比較兩者優缺點。 |

## 重要細節
- **Definitions:**
  - 絕對閾值 (Absolute threshold): Minimum audible signal.
  - 差異閾值 (Differential threshold): JND / Difference limen (DL).
  - 殘響 (Reverberation): 聲音在環境中多次反射的總和。
- **Mechanisms:**
  - 直達聲隨距離衰減，但殘響在空間中平均化後相對固定。因此「直達聲/殘響比例」是判斷距離的關鍵。
  - SDT 模型：大腦內部反應受雜訊影響呈現常態分布。有訊號與無訊號會產生兩個重疊的分布。d' 是兩分布平均值之差除以標準差。
- **Experiments / Psychophysical data:**
  - 1-interval 2-alternative task (yes-no): 易受 bias 影響。
  - 2-interval 2-alternative forced-choice (2AFC): 給兩個區間，一個有訊號一個沒有。減少 bias，以 percent correct 衡量。
  - Adaptive procedure (e.g., 2-down 1-up): 連對兩次降低強度，錯一次提高強度，最終收斂於 70.7% 的正確率。
- **Lecturer examples:**
  - 音樂後製中常利用調整殘響（wet/dry mix）來控制主唱與和聲的空間距離感。
  - 雷達控制員看螢幕、醫生看 X 光片找腫瘤，都屬於 SDT 的應用範疇。

## 現象與機制
1. **距離知覺 (Distance Perception)**
   - **Basis**: 物理聲學中距離與能量的關係，以及空間反射特性。
   - **Evidence/Scope**: 在無響室 (Anechoic room) 中難以判斷距離；音樂製作中調整殘響即可製造遠近錯覺。
   - **Connections**: 補足前一講水平與垂直定位（ITD, ILD, Pinna）以外的第三個空間維度。
2. **訊號偵測理論 (Signal Detection Theory, SDT)**
   - **Basis**: 神經系統的內部反應必然包含雜訊。
   - **Evidence**: 在臨界點附近的刺激，同一個刺激有時會聽見、有時聽不見。
   - **Scope**: 適用於所有偵測任務（聽覺、視覺、醫療診斷等）。
