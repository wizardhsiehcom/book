## 本講主問題
視覺系統接收到的輸入是平面的二維影像，但世界是三維的。大腦如何從二維影像中推導出物體的三維結構與深度？本講探討單眼深度線索（monocular depth cues），說明視覺系統如何結合物理與幾何知識的內隱假設（priors），將平面的光影、遮擋、透視與紋理等資訊轉譯為立體知覺。

## 核心概念

| 概念 | 解釋 | 處理方式 |
| --- | --- | --- |
| 深度知覺的適不良問題 (Ill-posed problem) | 從 2D 影像還原 3D 結構在數學上是多解的，必須依賴大腦內建的先驗假設來限制可能解。 | 貫穿全章作為核心理論框架。 |
| 由陰影推導形狀 (Shape from shading) | 基於朗伯表面 (Lambertian surface) 的特性，亮度變化可被詮釋為表面朝向的改變。大腦強烈預設「光從上方來」。 | 詳細介紹機制與翻轉影像的錯覺範例。 |
| 投射陰影 (Drop shadows) | 物體遮擋光源所產生的影子。能為物體錨定深度，即使局部陰影方向不一致，只要符合局部一致性就能產生深度感。 | 探討陰影如何讓物體看起來懸浮或貼地。 |
| 紋理梯度與視野高度 (Texture gradient & Height in field) | 視野上方的事物通常位於地面遠處，因此在視網膜的投影較小。紋理變密集與縮小暗示了深度的退行。 | 以平地彩繪成曲面的錯覺為例說明。 |
| 艾默特定律 (Emmert's law) | 知覺大小、知覺距離與視角之間的數學關係：相同的視角下，若知覺距離越遠，則知覺大小越大。 | 寫入核心概念，結合視覺後像實驗說明。 |
| 空氣透視 (Aerial perspective) | 遠處物體因光線在大氣中散射而顯得模糊、對比度低且偏藍。 | 簡要列為深度線索之一。 |
| 線性透視 (Linear perspective) | 現實中平行的線條在 2D 投影上會匯聚於一點。結合艾默特定律可產生怪物追逐等大小錯覺。 | 詳細說明透視錯覺的應用。 |
| 雙穩態知覺 (Bistability) | 當 2D 影像對應兩個機率相近的 3D 解釋時（如奈克方塊），知覺會在兩者間切換，無法同時看見。 | 放於章末討論推論的多模態分佈。 |

## 重要細節
1. **機制 - 朗伯表面 (Lambertian surface)**：反射光量由入射光與表面法向量的夾角決定，且光向各方向均勻散射。這是 shape from shading 的物理基礎。
2. **現象 - 光源先驗 (Prior of light from above)**：把月球隕石坑的相片倒過來會看成凸起的丘陵；倒轉沙丘相片則變成隕石坑。這種由陰影推論形狀的過程是認知不可穿透的（cognitively impenetrable）。
3. **實驗 - 視覺後像與艾默特定律**：盯著天花板的燈看產生後像（在視網膜上佔據固定視角），接著看近處的手會覺得後像很小，看遠處的牆會覺得後像變得巨大。
4. **錯覺 - 大怪物追小怪物 (Ponzo illusion 變體)**：兩隻在 2D 影像上大小完全相同的怪物，因為線性透視背景的關係，上面那隻看起來距離較遠，因而感覺巨大。
5. **錯覺 - Shepard 的平行四邊形桌面**：兩個在 2D 影像上形狀、大小完全相同的平行四邊形，因為桌腳與角度的提示，大腦會推論出截然不同的 3D 桌面形狀，且幾乎無法靠意識看出它們 2D 形狀相同。說明我們看到的是「推論出的 3D 結構」而非原始 2D 影像。
6. **局部一致性而非全域建模**：在多個球體與陰影的展示中，大腦並未建立一個完美的全域光源模型（否則會發現不同區域的影子暗示了不同光源），只要局部陰影與球體相對位置合理，就能產生深度。

## 現象與機制

- **單眼深度線索 (Monocular cues)**：
  - **形狀來自陰影 (Shape from shading)**：大腦預設光源來自上方，將漸層解析為凹凸。
  - **投射陰影 (Drop shadows)**：暗色的影子可定義物體相對表面的位置；影子移位會讓物體像在飄浮。
  - **視野高度與紋理**：越靠近視野上方、紋理越細密，代表距離越遠。
  - **空氣透視**：大氣散射導致遠物模糊偏藍。
  - **線性透視**：平行線匯聚，大腦據此推論深度。

- **艾默特定律 (Emmert's law)**：
  - **基礎**：視角大小（Visual angle）＝ 物理大小（Size）／ 距離（Distance）。
  - **機制**：如果我們知道物體的熟悉大小，就能推算距離；如果我們從深度線索得知了距離，就能推算物體的實際大小。
