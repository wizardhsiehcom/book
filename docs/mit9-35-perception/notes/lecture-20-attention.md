## 本講主問題
本講探討「注意力」在知覺處理中的角色與測量方法。講者介紹了各種經典的實驗典範，包括空間提示（Posner cueing）、多重物件追蹤（MOT）、聽覺串流（auditory streaming）以及視覺搜尋（visual search），旨在說明注意力如何幫助我們在複雜環境中選擇性地處理資訊，以及不同類型的注意力（如由外而內與由內而外）在處理時間與機制上的差異。

## 核心概念

| Concept | Explanation | How to handle in chapter |
|---|---|---|
| 注意力作為聚光燈 (Spotlight Metaphor) | 注意力就像一盞聚光燈，可以移動到空間中的特定位置，提升該區域的資訊處理速度與準確度。 | 在「機制與現象」中介紹，並透過 Posner 提示實驗來說明。 |
| 外源性與內源性提示 (Exogenous vs. Endogenous Cues) | 外源性提示是由外部刺激（如閃爍）自動引發的（Bottom-up）；內源性提示是由符號（如箭頭）引發，需要高階認知解讀與控制（Top-down）。 | 在核心概念中詳細區分兩者的差異，尤其是發生時間（SOA）的不同。 |
| 多重物件追蹤 (Multiple Object Tracking, MOT) | 證明注意力不只是單一聚光燈，人類有能力同時追蹤 4-5 個獨立移動的目標。 | 作為一個獨立小節，討論注意力資源的分配與限制。 |
| 聽覺注意力與雞尾酒會效應 (Auditory Attention) | 在多重相似的聲音來源中追蹤單一語音（透過追蹤 F0 與共振峰），成功追蹤者對該聲音的細節（如抖音 vibrato）更敏感。 | 結合之前聽覺章節的雞尾酒會效應，強調注意力在特徵空間中的連續追蹤。 |
| 視覺搜尋與特徵整合論 (Visual Search & FIT) | 區分特徵搜尋（Feature search，產生跳出效應 pop-out）與交集搜尋（Conjunction search，需要序列搜尋）。注意力被認為是將不同特徵（如顏色與形狀）綁定在一起的黏著劑。 | 詳細解釋反應時間與集合大小（set size）的關係，並介紹錯覺結合（Illusory conjunctions）。 |

## 重要細節
- **Definitions**: 
  - 外源性提示 (Exogenous cues)：Outside-generating，自動、反射性的。
  - 內源性提示 (Endogenous cues)：Inside-generating，需要理解符號並主動控制。
  - SOA (Stimulus Onset Asynchrony)：提示出現到目標物出現之間的時間差。
  - 集合大小 (Set size)：視覺搜尋實驗中干擾物的數量。
- **Mechanisms**: 注意力的由下而上（bottom-up）與由上而下（top-down）控制；特徵整合（Feature Integration）。
- **Experiments**: 
  - Posner Cueing 實驗（測量反應時間）。
  - Pylyshyn 的多重物件追蹤（MOT）實驗。
  - 聽覺母音追蹤與抖音（vibrato）偵測實驗。
  - Treisman 的視覺搜尋與錯覺結合（Illusory conjunctions）實驗。
- **Psychophysical data**: 
  - Posner 實驗中，有效提示（valid cue）可減少反應時間。
  - MOT 實驗中，追蹤 1-5 個物件時錯誤率隨數量上升，但仍維持在可接受範圍。
  - 視覺搜尋實驗：特徵搜尋斜率為 0（Pop-out）；交集搜尋斜率大於 0，且目標缺席（Target absent）的斜率大約是目標存在（Target present）的兩倍。
- **Lecturer examples**: 棒球投手與捕手的例子（結合外源性與內源性提示）；找鑰匙、找水龍頭的日常交集搜尋例子。
- **Q&A highlights**: 
  - 提問：外源性與內源性提示是否為二分法？講者回答：日常生活中常是兩者結合（例如看棒球，知道球會進好球帶是內源性，聽到捕手手套聲音是外源性）。
  - 提問：MOT 能力可以訓練嗎？講者回答：有研究指出籃球運動員或電玩玩家在這類視覺注意力任務上表現較好，但不確定是訓練造成還是天擇。
  - 提問：為什麼視覺搜尋目標缺席的斜率是兩倍？學生回答：因為目標存在時平均只需找一半，目標缺席時必須全部找完（exhaustive search）。

## 現象與機制
- **機制：特徵跳出 (Pop-out)**
  - Basis: 單一明顯特徵差異。
  - Evidence: 反應時間與集合大小無關（平坦斜率）。
- **現象：錯覺結合 (Illusory conjunctions)**
  - Basis: 當注意力資源不足（或被轉移到中央任務）時，特徵（如顏色與形狀）無法正確綁定。
  - Evidence: 受試者會報告看到「紅色的 X」和「藍色的 T」，即使實際出現的是「藍色的 X」和「紅色的 T」。
