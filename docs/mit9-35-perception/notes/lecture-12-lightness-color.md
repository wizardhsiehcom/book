## 本講主問題
本講接續探討視覺系統如何從視網膜接收到的光線中，估計物體表面的反射率（即亮度和顏色）。首先分析了分離光照與反射率的歷史與計算模型（Retinex 理論）及其侷限，並展示中階視覺如何利用局部邊界線索解決光照問題。隨後，課程轉入顏色知覺，介紹光譜物理學、人類三色視覺（Trichromacy）的生理限制（單一變量原則與同色異譜），以及如何與對立色彩理論（Opponent-Process Theory）相統一，展示神經系統對無限維度光譜信號的降維處理。

## 核心概念

| 概念 | 解釋 | 處理方式 |
| :--- | :--- | :--- |
| **Lightness Constancy (亮度恆常性)** | 人類能在不同光照條件下正確估計物體表面反射率的能力，是視覺系統試圖解決的不適定問題（Ill-posed problem）。 | 在章節首段定義問題背景（L = I × R）。 |
| **Retinex Theory** | Edwin Land 提出的算法。利用真實世界的統計規律假設：光照變化多為平緩，而反射率變化多為劇烈邊緣。透過對圖像取空間導數並整合，來分離兩者。 | 作為計算層次與算法層次的經典範例進行詳細講解，並附帶推導邏輯。 |
| **Stimulus Computability (刺激可計算性)** | 理論或算法明確到可以輸入任意真實圖像並輸出預測，這使得 Retinex 等理論可以被實際檢驗並找出缺陷。 | 強調這是感知科學和計算機視覺模型的重要標準。 |
| **Mid-level Illumination Cues (中階局部光照線索)** | 人類視覺系統似乎不完全依賴高階完整的 3D 幾何模型來折扣光照，而是依賴如邊界、T型/X型接點、陰影邊緣（Penumbra）等局部啟發式規則（Heuristics）。 | 透過 Snake illusion 和 Tips and strips illusion 來展示其效應。 |
| **Trichromacy (三色視覺)** | 人類視網膜包含三種視錐細胞（S, M, L），將無限維度的連續光譜降維為三個數值。 | 解釋波長、光譜靈敏度曲線，以及降維帶來的數學本質。 |
| **Principle of Univariance (單一變量原則)** | 光子一旦被吸收，對感光細胞產生的反應就是一維的強度變化，細胞無法區分光源的波長與強度差異。 | 這是產生同色異譜和需要三種視錐細胞的根本生理原因。 |
| **Metamers (同色異譜)** | 物理光譜截然不同，但刺激視錐細胞後產生完全相同三維反應組合的兩種光，在感知上無法區分。 | 舉例顯示鎢絲燈與螢幕像素的匹配，說明其在顯示科技上的應用。 |
| **Opponent Color Theory (對立色彩理論)** | Hering 提出，基於紅綠、藍黃的排斥現象及色盲成對發生。與三色理論在不同神經層次上達成統一。 | 在章節末段介紹，說明這發生在視網膜神經節細胞層級。 |

## 重要細節
- **歷史爭論：** Helmholtz 的「無意識推論（Unconscious inference，心理學/高階層次）」與 Hering 的「神經機制（生理學/底層網格）」，兩人對同時對比（Simultaneous contrast）有不同解釋，但後來發現兩者並非互斥。Marr 將其歸納為多層次分析。
- **錯覺案例：** 
  - Craik-O'Brien-Cornsweet Effect：平緩漸層與銳利邊緣組合，支持了大腦高度依賴邊緣（導數大）來判斷亮度差異的假說（與 Retinex 相符）。
  - Snake Illusion / Tips and strips：展示當光照邊界非常銳利時，Retinex 算法會失敗（錯誤歸類為反射率變化），但人類卻能正確折扣光照。
  - 3D 模型無效性證明：某些改變了 3D 解釋但保留了局部 2D 幾何線索的錯覺圖形，依然產生相同的亮度錯覺，證明視覺系統偏好中階（Mid-level）局部線索。
- **物理與心理物理學：**
  - 在暗視（Scotopic）條件下，僅有視桿細胞運作，其靈敏度曲線與視紫質吸收曲線完全吻合，驗證了單一變量原則。
  - 色彩空間可以用線性代數（矩陣相乘）來完美理解：從無限維光譜向量，經過包含三條敏感度曲線的矩陣，投影為三維空間的點。

## 現象與機制
- **Retinex 機制：**
  - 基礎：光照在空間中變化平緩（導數小），反射率變化劇烈（導數大）。
  - 證據：對某些邊緣對比錯覺（如 Cornsweet）解釋良好。
  - 範圍與缺陷：無法處理陰影造成的銳利邊緣（會被誤認為反射率），也無法處理因 3D 形狀導致的平緩亮度變化（會被誤認為光照）。
- **Metamers (同色異譜) 機制：**
  - 基礎：單一變量原則與三色視覺的降維投影。
  - 證據：人們可以透過調整紅藍綠三個光源的強度，完美匹配出與任何連續光譜物理光源在感知上相同的顏色。
  - 應用：彩色螢幕、相機、底片。
