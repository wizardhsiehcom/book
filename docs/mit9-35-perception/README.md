# MIT 9.35《Sensation and Perception 感覺與知覺》

本書根據 MIT 9.35 課程逐字稿整理，是一本以繁體中文撰寫的感覺與知覺系統入門書。課程由 Josh McDermott 教授主講，共 23 講，涵蓋聽覺、視覺（從低階到高階）、注意力，以及觸覺、嗅覺與味覺。

## 讀者對象

本書適合對認知科學、神經科學或心理學有興趣，想要系統理解人類感覺與知覺運作機制的讀者。假設讀者具備基本的生物與物理常識，但不預設神經科學背景。
