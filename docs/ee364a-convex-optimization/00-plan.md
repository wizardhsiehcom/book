# 全書地圖

本書以 EE364a 的 18 份逐字稿（Boyd 2023）為主資料來源。每一章必須先完成完整閱讀筆記，再寫成書稿。

## 暫定篇章安排

下表為依教科書結構做的**暫定**分篇。18 講與教科書 13 章不必一對一，各講實際主題以完整閱讀確認後回填、調整。

| 篇 | 暫定章節 | 主題 |
|---|---|---|
| 第一篇：凸優化基礎 | Lecture 1-4 | 導論、凸集、凸函數、凸優化問題的辨識 |
| 第二篇：對偶理論 | Lecture 5-6 | Lagrange 對偶、弱／強對偶、KKT 條件 |
| 第三篇：應用問題 | Lecture 7-9 | 逼近與擬合、統計估計、幾何問題 |
| 第四篇：演算法基礎 | Lecture 10-13 | 數值線性代數背景、無約束與等式約束最小化 |
| 第五篇：內點法與收尾 | Lecture 14-18 | 內點法與課程總結、延伸主題 |

> 篇章與講次對應為暫定骨架；每一講讀完後回填實際講題再調整。

## 寫作流程

1. 完整讀完單講逐字稿。
2. 寫閱讀筆記，記錄主問題、概念、定義／定理、板書推導與跨章連結。
3. 將筆記改寫成章節初稿。
4. 所有逐字稿初稿完成後，再搜尋外部理解與補充資料。
5. 最後統一術語、交叉引用、圖表與參考資料格式。
