## 本講主問題

本講探討如何有效率地求解無約束的平滑凸優化問題。我們發現梯度下降法會受到問題條件數（次級集形狀）的嚴重影響，因此引入了能適應函數局部曲率的牛頓法（Newton's Method）。本講旨在介紹牛頓步的幾何與代數意義、仿射不變性，以及其實際收斂行為與理論分析（包括傳統分析與自和諧分析）的差異。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| 最速下降法 (Steepest Descent) | 取決於所選擇的範數。在特定的二次範數下，最速下降方向會改變，以適應不同的次級集形狀。 | 在前導段落介紹，說明範數選擇的重要性。 |
| 牛頓步 (Newton Step) | 可以解釋為最小化函數的二次近似，或在由 Hessian 定義的局部度量下的最速下降方向。 | 作為核心算法詳細介紹。 |
| 牛頓遞減量 (Newton Decrement) | 衡量當前點到最優點的接近程度，是仿射不變的，也是停止準則的基礎。 | 詳細定義公式並說明其物理意義（預測的函數下降量）。 |
| 仿射不變性 (Affine Invariance) | 牛頓法不受仿射座標轉換的影響，這意味著它對問題的縮放不敏感。 | 強調這是牛頓法極具實用價值的原因。 |
| 傳統收斂分析 | 依賴不可知的參數（如 Lipschitz 常數 $L$、強凸參數 $m$），且分析結果不具備仿射不變性，常被認為理論意義大於實用價值。 | 作為背景知識簡述，並指出其理論與實際的脫節。 |
| 自和諧 (Self-concordance) | 由 Nesterov 和 Nemirovski 提出，利用三階導數受限於二階導數的特性，給出完全仿射不變的收斂分析。 | 用來解釋為什麼某些問題可以用仿射不變的方式給出嚴格的收斂界限。 |
| 結構化 Hessian 與線性代數求解 | 牛頓法將平滑優化轉換為解一系列線性方程組。如果 Hessian 具備稀疏性或帶狀結構，可以極大降低運算成本。 | 在實作考量段落重點探討，連結到上一講的數值線性代數。 |

## 重要細節

- 定義：最速下降方向、牛頓步、牛頓遞減量、自和諧函數。
- 定理／性質：牛頓法具備二次收斂性（Quadratic Convergence）；仿射不變性。
- 公式：$H \Delta x = -g$ (牛頓步解法), $\lambda(x) = (g^T H^{-1} g)^{1/2}$ (牛頓遞減量)。
- 演算法：阻尼牛頓法（Damped Newton Phase）與二次收斂階段（Quadratically Convergent Phase）。
- 板書推導：無明顯板書推導，主要是投影片講解。
- 講者例子與幾何直覺：在狹長的次級集中，梯度下降會呈現 zigzagging（鋸齒狀）前進；而牛頓法透過 Hessian 「拉平（round）」問題形狀，使其像在圓形次級集中最佳化。
- 應用場景：邏輯斯迴歸（最大概似估計中 Fisher Information 作為 Hessian）、平滑控制與訊號處理問題（帶狀 Hessian）。
- 問答重點：學生提問如果無法計算 Hessian 怎麼辦（引出 Quasi-Newton 方法和自動微分）。
- 容易忽略的提醒：理論上的迭代次數上限通常大得不切實際，實務上牛頓法通常在極少次迭代（如 10~20 次）內即可收斂。

## 求解與建模的意義

說明這一講對「辨識並求解凸優化問題」的實際幫助：

- 這一講讓你能辨識／建模什麼問題：這講主要不是建模，而是告訴我們，只要是平滑凸優化問題，且能計算（或利用結構有效求解）Hessian，就可以用牛頓法極快求解。

- 會影響哪些後續章節：為後續的等式約束牛頓法（Equality Constrained Newton Method）以及內點法（Interior Point Methods）奠定基礎。
