# 組織與治理

台積電採用矩陣式組織結構，研發、製造、業務三條線平行運作，並由最高技術委員會統籌技術方向。

## 獨特的企業文化

台積電以**誠信（Integrity）** 為核心價值，具體體現在：

- **不與客戶競爭**：台積電不自行設計或銷售晶片，確保客戶設計圖不外洩
- **高度保密文化**：員工不得討論特定客戶或製程細節
- **長期技術投資**：研發支出佔營收比例穩定維持約 8–9%
- **嚴格的品質管控**：良率（Yield）是衡量廠區績效的核心指標
