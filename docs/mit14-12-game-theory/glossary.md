# 術語表 (Glossary)

全書中英文對照與解概念整理於此。

## 解概念與賽局分類 (Solution Concepts & Game Types)

| 英文 | 中文 | 首次出現章節 | 說明 |
|---|---|---|---|
| Dominance / Strict Dominance | 優勢 / 嚴格優勢 | 第 3 章 | 策略在任何對手選擇下皆帶來較高報酬 |
| Rationalizability | 可理性化 | 第 4 章 | 在共同理性知識下可能被選用的策略集合 |
| Nash Equilibrium (NE) | 納許均衡 | 第 5 章 | 所有玩家同時對其他人的策略做出最佳反應 |
| Backward Induction | 逆向歸納法 | 第 8 章 | 從賽局樹末端往回推導完美資訊動態賽局解的方法 |
| Subgame-Perfect Nash Equilibrium (SPNE) | 子賽局完美納許均衡 | 第 10 章 | 均衡策略在每一個子賽局中都構成納許均衡 |
| Bayesian Game | 貝氏賽局 | 第 16 章 | 包含不完全資訊（玩家擁有私人型態）的賽局 |
| Bayesian Nash Equilibrium (BNE) | 貝氏納許均衡 | 第 16 章 | 貝氏賽局中的納許均衡，根據先驗機率做期望值極大化 |
| Perfect Bayesian Equilibrium (PBE) | 完美貝氏均衡 | 第 21 章 | 結合 SPNE 序列理性與貝氏定理信念更新的動態解概念 |
| Zero-Sum Games | 零和賽局 | 第 7 章 | 玩家之間利益完全對立的賽局 |
| Finitely / Infinitely Repeated Games | 有限 / 無限重複賽局 | 第 12/13 章 | 同一階段賽局被重複進行有限或無限次 |
