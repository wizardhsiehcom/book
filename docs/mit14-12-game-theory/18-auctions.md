# 第 18 講：拍賣 (Auctions) 一般化模型

## 導讀

在上一講中，我們透過一個簡單的雙人拍賣範例（兩人的估價皆服從 $[0,1]$ 的均勻分佈），計算了第一價格拍賣（First-Price Auction, FPA）與第二價格拍賣（Second-Price Auction, SPA）的均衡結果。本講將進一步推廣，探討更一般的「對稱獨立私有價值」（Symmetric Independent Private Values, SIPV）模型。我們將嚴格證明在第二價格拍賣中「誠實出價」為何是弱優勢策略，接著推導一般化第一價格拍賣的貝氏納許均衡（BNE），並初步揭示拍賣理論中最優美的結果之一：收益等價定理（Revenue Equivalence Theorem）。

## 核心內容

### 簡單均勻分佈模型的收益回顧

回顧上一講的例子，有兩位競標者，其估價 $v_1, v_2$ 獨立且均勻分佈於 $[0,1]$。
1. **第一價格拍賣 (FPA)**：我們找到了一個對稱均衡，每個人出價為其估價的一半，$b_i = v_i / 2$。
2. **第二價格拍賣 (SPA)**：每個人出價等於其真實估價，$b_i = v_i$。

拍賣官（假設為風險中立且只在乎期望收益最大化）會偏好哪一種拍賣機制呢？
- 在 FPA 中，拍賣官獲得最高的出價：$\max(b_1, b_2) = \max(v_1/2, v_2/2) = \frac{1}{2} \max(v_1, v_2)$。因 $\max(v_1, v_2)$ 的期望值為 $2/3$，故期望收益為 $1/3$。
- 在 SPA 中，拍賣官獲得第二高的出價，也就是 $\min(b_1, b_2) = \min(v_1, v_2)$。其期望值剛好也是 $1/3$。

這裡存在一個微妙的權衡：在 FPA 中，拍賣官收取「最高出價」，但競標者為了獲利會「壓低出價」（bid shading）；在 SPA 中，拍賣官只收取「第二高出價」，但競標者會「誠實出價」。在這兩個特例中，這兩種力量完美抵銷，導致預期收益完全相同。這絕非巧合。

### 第二價格拍賣中的誠實出價證明

我們可以用圖解法嚴格論證，為何在 SPA 中，出價等於真實估價 $v_i$ 是「弱優勢策略」（Weakly Dominant Strategy）。也就是說，不論對手如何出價，出價 $v_i$ 永遠不會比其他出價帶來更差的回報。

假設我是競標者 1，真實估價為 $v_1$。考慮偏離到較高的出價 $b_1' > v_1$：
我們比較出價 $v_1$ 與 $b_1'$，並根據對手的最高出價 $b_2$ 落在何處分為三種情況：
1. **$b_2 < v_1$**：無論我出價 $v_1$ 或 $b_1'$，我都會贏得拍賣，且支付的金額都是 $b_2$。這兩種出價策略結果無差異。
2. **$b_2 > b_1'$**：無論我出價 $v_1$ 或 $b_1'$，我都會輸掉拍賣，支付 0，效用 0。兩者無差異。
3. **$v_1 < b_2 < b_1'$**（關鍵區間）：如果我出價 $v_1$，我會輸掉拍賣，效用 0。如果我出價 $b_1'$，我會贏得拍賣，但我必須支付第二高出價 $b_2$。因為 $b_2 > v_1$，我支付的金額大於我的真實估價，導致**負的利潤**。因此，出價 $v_1$ 嚴格優於出價 $b_1'$。

同理，若考慮偏離到較低的出價 $b_1' < v_1$，在 $b_1' < b_2 < v_1$ 的區間，出價 $b_1'$ 會讓你輸掉一場本來可以贏得且利潤為正的拍賣。因此，誠實出價在 SPA 中是弱優勢策略。

## 形式化與定理：一般化第一價格拍賣

為了突破均勻分佈特例，我們引入**對稱獨立私有價值 (SIPV) 模型**：
- $n$ 個競標者。
- **對稱 (Symmetric)**：每個人的估價 $v_i$ 從相同的分佈抽取，落於 $[\underline{v}, \overline{v}]$ 之間，累積分配函數為 $F(v)$，機率密度函數為 $f(v)$。
- **獨立 (Independent)**：得知自己的估價無法推論別人的估價。
- **私有價值 (Private Values)**：每個人清楚知道自己的估價，且估價不受他人估價的影響（例如不是要轉售的藝術品）。

### 求解 BNE

我們要尋找第一價格拍賣的貝氏納許均衡。假設存在一個**對稱、嚴格遞增且可微**的出價函數 $\beta(v)$，所有競標者都使用這個策略。

令 $G(v)$ 為「$n-1$ 個對手中最高估價」的 CDF，即 $G(v) = (F(v))^{n-1}$。其密度函數為 $g(v) = G'(v)$。

對於競標者 1（估價 $v_1$），假設所有對手皆使用 $\beta(\cdot)$。若他出價 $b_1$，其期望效用為：
$$U_1(b_1 \mid v_1) = (v_1 - b_1) \times \mathbb{P}(b_1 > \text{對手的最高出價})$$

因為對手出價為 $\beta(v_j)$ 且 $\beta$ 遞增，對手最高出價對應於對手的最高估價。因此獲勝條件為 $b_1 > \beta(\max_{j \neq 1} v_j)$，也就是 $\max_{j \neq 1} v_j < \beta^{-1}(b_1)$。機率即為 $G(\beta^{-1}(b_1))$：
$$U_1(b_1 \mid v_1) = (v_1 - b_1) G(\beta^{-1}(b_1))$$

對 $b_1$ 微分求一階條件 (FOC)：
$$\frac{\partial U_1}{\partial b_1} = -G(\beta^{-1}(b_1)) + (v_1 - b_1) g(\beta^{-1}(b_1)) \frac{1}{\beta'(\beta^{-1}(b_1))} = 0$$

在對稱均衡下，最佳出價必須恰好是 $b_1 = \beta(v_1)$。代入並整理後得到微分方程：
$$\beta'(v_1) G(v_1) + \beta(v_1) g(v_1) = v_1 g(v_1)$$

利用乘積法則（Product Rule），等式左邊剛好是 $\frac{d}{dv} [\beta(v) G(v)]$。從下界 $\underline{v}$ 積分至 $v$（且因 $G(\underline{v})=0$）：
$$\beta(v) G(v) = \int_{\underline{v}}^v x g(x) dx$$
$$\beta(v) = \frac{1}{G(v)} \int_{\underline{v}}^v x g(x) dx$$

### 條件期望值的直覺詮釋

這條公式有一個非常直覺的統計詮釋。積分部分除以 $G(v)$，正好是條件期望值：
$$\beta(v) = \mathbb{E}\left[\max_{j \neq 1} v_j \mid \max_{j \neq 1} v_j \le v\right]$$
亦即，在 FPA 中的均衡出價等於：**「假設我的估價比所有對手都高（我會贏得拍賣）的條件下，預期對手中最高的估價是多少。」**

## 收益等價定理 (Revenue Equivalence) 初探

讓我們比較一個擁有估價 $v$ 的競標者在兩種拍賣制下的**期望支付（Expected Payment）**：
- 在 **第一價格拍賣 (FPA)**：只要他贏了，他就必須支付他的出價 $\beta(v)$。由於出價沒有隨機性，給定他贏得拍賣，其支付金額確定為 $\mathbb{E}\left[\max_{j \neq 1} v_j \mid \max_{j \neq 1} v_j \le v\right]$。
- 在 **第二價格拍賣 (SPA)**：他誠實出價 $v$。如果他贏了，他需支付第二高出價（即對手的最高估價 $\max_{j \neq 1} v_j$）。這個支付金額是隨機的，但「給定他贏得拍賣」的條件下，其**預期支付金額**恰好也是 $\mathbb{E}\left[\max_{j \neq 1} v_j \mid \max_{j \neq 1} v_j \le v\right]$。

這意味著，無論採用 FPA 還是 SPA，任何給定估價的競標者，其預期支付給拍賣官的金額完全相同。自然地，拍賣官從這兩種拍賣機制中獲得的總預期收益也完全一樣。這就是「收益等價定理」的核心概念。

## 常見誤解

- **出價函數的機率混淆**：在計算預期效用時，經常有人把自己的出價 $b_1$ 當作隨機變數。實際上，$b_1$ 是一個我們（作為決策者）要選擇的確定數值；隨機性來自於我們不知道對手的真實估價，因而不知道對手的實際出價。
- **邊界條件忽略**：解微分方程時，必須記得最低估價者贏得拍賣的機率為 0（$G(\underline{v}) = 0$），這讓積分常數得以確定，從而解出封閉形式的出價函數。

## 小結

本講將拍賣模型推廣到一般的 SIPV 設定。我們確認了第二價格拍賣中誠實出價的強健性，並利用微分方程找出了第一價格拍賣中，以對手最高估價之條件期望值為出價策略的對稱貝氏納許均衡。最後，我們發現兩種截然不同的機制卻能帶給拍賣官相同的期望收益，為接下來深入探討「收益等價定理」埋下了伏筆。
