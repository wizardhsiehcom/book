## 本講主問題

本講探討有限次數的重複賽局（Finitely Repeated Games）。重點在於分析當玩家有機會觀察過去的行動時，是否能透過未來的獎勵與懲罰機制來支持在單次賽局中無法達成的合作。講者介紹了當階段賽局只有唯一納許均衡時的定理，並透過獵鹿賽局（Stag Hunt）說明當存在多重納許均衡時，如何建構出依賴歷史的獎懲策略。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| Finitely Repeated Games (有限重複賽局) | 將同一個階段賽局（Stage Game）重複進行已知且有限的次數 $T$。 | 詳細定義並介紹其時間與資訊結構。 |
| Observability (可觀察性) | 每期賽局結束後，玩家能觀察到所有人的行動，這是建立獎勵與懲罰機制的先決條件。 | 在模型設定中強調此特徵。 |
| Unique Nash Equilibrium Theorem | 若階段賽局有唯一的納許均衡，則有限重複賽局具有唯一的子賽局完美納許均衡（SPNE），即每期皆無條件選擇該階段賽局的均衡。 | 敘述定理並以向後歸納法（Backward Induction）給出直覺證明。 |
| Stag Hunt (獵鹿賽局) | 具有兩個純策略納許均衡（Stag-Stag, Hare-Hare）的賽局，體現了合作與安全策略的拉扯。 | 列出收益矩陣並分析其多重均衡特性。 |
| Rewards and Punishments (獎勵與懲罰) | 當階段賽局有多重均衡時，可利用較好/較差的均衡作為未來行動的獎勵/懲罰，以支持當期的非均衡行動。 | 以 Stag Hunt 兩期賽局為例，具體建構依賴歷史的 SPNE 策略。 |
| One-shot Deviation Principle | 單次偏離原則，因為有限重複賽局屬於多階段賽局，可利用此原則簡化 SPNE 的檢驗。 | 說明在檢驗重複賽局策略時如何應用。 |

## 重要細節

- 定義（解概念／賽局元素／經濟量）：
  - 階段賽局 $G$、重複賽局 $G(T)$、歷史（History）$h_t$、平均收益（Average Payoff）。
- 定理敘述與證明思路：
  - 定理：若階段賽局有唯一純策略納許均衡 $a^*$，則在 $T$ 期重複賽局中，唯一的 SPNE 為在所有歷史下皆選擇 $a^*$。證明思路使用向後歸納法：最後一期因為沒有未來，只能玩階段賽局均衡；倒數第二期知道最後一期的結果與歷史無關，故也只能玩階段賽局均衡，依此類推。
- 賽局實例（矩陣、賽局樹、payoff 設定；標明「依口語重建」或 `待補`）：
  - 囚徒困境（Prisoner's Dilemma，依口語重建）：C, D 兩個行動。收益矩陣為 (C,C)=(2,2), (D,C)=(3,-1), (C,D)=(-1,3), (D,D)=(0,0)。
  - 獵鹿賽局（Stag Hunt，依口語重建）：S, H 兩個行動。收益矩陣為 (S,S)=(2,2), (H,H)=(1,1), (H,S)=(1,0), (S,H)=(0,1)。
- 課堂 MobLab 遊戲：規則、結果、講者詮釋：
  - 將囚徒困境重複玩 3 次（類似剪刀石頭布的形式）。許多學生選擇連續合作，少部分全選背叛。講者指出理論預測（SPNE）是永遠背叛，學生的合作行為可能反映了人類思考中將遊戲視為更大互動框架的一部分（Paul Milgrom 所說的「賽局總比你想的還大」）。
- 經濟應用場景：
  - 企業共謀（Firm collusion），透過未來懲罰來嚇阻當前的降價行為。
- 講者例子或直覺說明：
  - 在說明策略建構時，講者強調策略是「完整的應變計畫」，不僅能根據對手的行為，也能根據「自己」過去的行為來決定未來的行動。
- 問答重點：
  - 學生提問在囚徒困境中策略的定義，講者釐清策略與結果（Outcome）的不同，策略必須涵蓋所有可能的歷史節點。
- 容易忽略的提醒：
  - 重複賽局中的 payoff 常使用平均 payoff（除以 $T+1$），這樣能將數值規模與階段賽局保持一致，不影響誘因。

## 解概念與均衡分析

- 解概念名稱：Subgame Perfect Nash Equilibrium (SPNE)
- 形式化定義：在每個子賽局中，策略組合的限制（Restriction）都構成該子賽局的 Nash Equilibrium。
- 求解程序或演算法：Backward Induction（向後歸納法）與 One-shot Deviation Principle。
- 適用範圍與限制（本講講者實際說明者）：適用於有限重複賽局，強調若有獨特納許均衡則無合作空間，必須有多重納許均衡才能產生歷史相依的 SPNE。
- 在解概念精煉鏈上的位置與前後關聯：建構在 Nash Equilibrium 之上，透過動態的歷史與子賽局來排除不可置信的威脅（Incredible threats）。
