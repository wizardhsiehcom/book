## 本講主問題

本講旨在探討「無限期重複賽局（Infinitely Repeated Games）」。在沒有自然終點（last period）的設定下，排除了有限期重複賽局中因逆向歸納造成的「最後一期效應」，探討玩家是否能透過未來的獎懲機制（如冷酷觸發策略）來維持合作。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| Discount factor (折現因子 $\delta$) | 衡量未來收益相對於現在價值的比例 ($\delta \in (0,1)$)。可解釋為利率、遊戲繼續的機率、或純粹的跨期不耐。 | 在定義無限期報酬時引入並詳述其三種詮釋。 |
| Discounted payoffs | 無限期賽局中，因期數無限無法直接加總，故使用折現加總 $\sum_{t=0}^{\infty} \delta^t u_i(a^t)$ 來計算報酬。 | 寫出數學式，並說明如何提取常數項來簡化子賽局的效用比較。 |
| Grim Trigger Strategy (冷酷觸發策略) | 只要歷史中無人背叛就合作；一旦有人背叛，就永遠背叛。是一種極度嚴厲的懲罰機制。 | 寫出形式化策略條件，並計算其成為 SPNE 的 $\delta$ 門檻。 |
| Tit-for-Tat (以牙還牙策略) | 第一期合作，之後每一期都模仿對手前一期的行動。 | 說明其行為結果雖為永遠合作，但在絕大多數 $\delta$ 下皆非 SPNE。 |

## 重要細節

- 定義（解概念／賽局元素／經濟量）：折現因子 $\delta$、無限期歷史 $\mathcal{H}$、無限期策略函數。
- 定理敘述與證明思路：利用 One-Shot Deviation Principle（單次偏離原則）檢驗 SPNE，將無限多個歷史依據策略特徵分組（例如：有人背叛過 vs 無人背叛過），以簡化驗證過程。
- 賽局實例（矩陣、賽局樹、payoff 設定；標明「依口語重建」或 `待補`）：
  - 囚徒困境（Prisoner's Dilemma, 依口語重建）：C(2,2), D vs C (3,-1), C vs D (-1,3), D(0,0)。
- 經濟應用場景：企業定價、政府談判（沒有明確的終結日）。
- 講者例子或直覺說明：解釋為何冷酷觸發策略中，一旦自己背叛，下一期自己也會選擇背叛（因為預期對手會懲罰，所以自己也乾脆背叛）。

## 解概念與均衡分析

- 解概念名稱：Subgame Perfect Nash Equilibrium (SPNE)
- 形式化定義：在任何可能的歷史 $h$（即使是均衡路徑外）構成的子賽局中，策略組合都必須是納許均衡。
- 求解程序或演算法：
  1. 定義策略（對所有歷史規定行動）。
  2. 將歷史分組（例如：包含 D 的歷史、不包含 D 的歷史）。
  3. 對每一組歷史，應用單次偏離原則（One-Shot Deviation Principle），比較「遵循策略的未來折現總和」與「單次偏離後再遵循策略的未來折現總和」。
  4. 找出使偏離無利的 $\delta$ 條件。
- 適用範圍與限制（本講講者實際說明者）：因為引入折現因子，這些賽局滿足「連續性（continuity）」的技術條件，所以單次偏離原則可以適用。
