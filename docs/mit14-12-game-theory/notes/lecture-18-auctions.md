## 本講主問題

本講旨在將前述的簡單雙人均勻分佈賽局擴展，探討更一般化的拍賣模型（SIPV 模型，對稱獨立私有價值）。並透過求解第一價格拍賣（FPA）的一般化貝氏納許均衡，以及比較 FPA 與第二價格拍賣（SPA）的期望收益，引入拍賣理論中著名的「收益等價定理」（Revenue Equivalence Theorem）。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| 第二價格拍賣 (SPA) 中誠實出價是弱優勢策略 | 無論對手如何出價，出價等於真實估價永遠不會比其他出價帶來更差的結果。 | 利用圖解（分三區間）詳細證明向上/向下偏離為何不具優勢。 |
| 對稱獨立私有價值模型 (SIPV) | $n$ 個競標者，估價從相同分佈 $F(v)$ 獨立抽取；且每個人清楚知道自己的估價，且估價不取決於他人。 | 形式化定義並解說其假設的涵義與限制。 |
| FPA 的均衡出價策略 | 猜測對稱、嚴格遞增且可微的出價函數 $\beta(v)$，利用一階條件解出微分方程。 | 詳細推導 $U_1(b_1) = (v_1 - b_1) G(\beta^{-1}(b_1))$ 的極大化過程與隱函數定理的應用。 |
| 均衡出價的條件期望值詮釋 | FPA 的均衡出價等於「在對手最高估價小於我的估價之條件下，對手最高估價的期望值」。 | 用公式 $\beta(v) = E[\max_{j \neq 1} v_j \mid \max_{j \neq 1} v_j \le v]$ 解釋直覺。 |
| 收益等價定理 (RET) 初探 | 儘管 FPA 和 SPA 的出價策略不同，但給定估價 $v$ 下，得標時的期望支付（expected payment）在這兩種拍賣制下完全相同。 | 以條件期望值說明 FPA 必定支付 $\beta(v)$，而 SPA 支付隨機的對手最高出價，兩者期望值相等。 |

## 重要細節

- 定義（解概念／賽局元素／經濟量）：
  - $F(v)$: 估價的累積分配函數 (CDF)。
  - $f(v)$: 估價的機率密度函數 (PDF)。
  - $G(v) = F(v)^{n-1}$: $n-1$ 個對手中最高估價的 CDF。
  - $g(v) = (n-1)F(v)^{n-2}f(v)$: $G(v)$ 的 PDF。
- 定理敘述與證明思路：
  - 證明 SPA 誠實出價是弱優勢策略：考慮偏離到 $b' > v$ 或 $b' < v$。對於對手出價 $b_2$ 落在 $v$ 與 $b'$ 之間的情況，偏離會導致贏得拍賣但支付超過估價（利潤為負），或者輸掉本可贏得並獲利的拍賣。
  - 解 FPA 微分方程：得到一階條件 $\beta'(v)G(v) + \beta(v)g(v) = v g(v)$。利用乘積法則反向積分，解得 $\beta(v) = \frac{1}{G(v)} \int_{\underline{v}}^v x g(x) dx$。
- 賽局實例（矩陣、賽局樹、payoff 設定；標明「依口語重建」或 `待補`）：
  - 均勻分佈 [0,1] 的 2 人拍賣回顧：
    - FPA: 均衡出價 $b_i = v_i / 2$。拍賣官期望收益 $\frac{1}{2} E[\max(v_1, v_2)] = 1/3$。
    - SPA: 均衡出價 $b_i = v_i$。拍賣官期望收益 $E[\min(v_1, v_2)] = 1/3$。
  - 圓環上的均勻分佈（幾何直覺）：將 [0,1] 線段彎成圓，3 個點（包含原點與兩隨機變數）期望等距切分，故兩變數的較大值期望為 2/3，較小值期望為 1/3。
- 課堂 MobLab 遊戲：無。
- 經濟應用場景：拍賣設計（拍賣官選擇何種拍賣制度可最大化期望收益）。
- 講者例子或直覺說明：
  - 解釋 $\beta'(v)G(v) + \beta(v)g(v) = v g(v)$ 直覺：稍微提高出價的邊際效益（以機率 $g(v)\epsilon$ 多贏得拍賣，獲利 $v - \beta(v)$）與邊際成本（在 $G(v)$ 的贏得機率下多支付 $\beta'(v)\epsilon$）必須相等。
- 問答重點：
  - 學生提問：為何 G(x) 會變成條件期望？解釋條件機率密度的定義 $g(x \mid x \le v) = \frac{g(x)}{G(v)}$。
- 容易忽略的提醒：
  - $G(v)$ 的定義是對手（共 $n-1$ 人）中的最大值，而非所有人中的最大值。
  - 計算出價的函數時，需確認該出價策略使得對手最佳反應為遵循該函數（對稱均衡）。

## 解概念與均衡分析

- 解概念名稱：Bayesian Nash Equilibrium (BNE) / Weakly Dominant Strategy
- 形式化定義：SPA 中出價真實估價為弱優勢；FPA 中尋求對稱、嚴格遞增且可微的均衡。
- 求解程序或演算法：
  1. 寫出給定自己估價 $v_1$ 與出價 $b_1$ 下的預期效用。
  2. 將對手出價代入猜測的均衡函數 $\beta(v)$。
  3. 對 $b_1$ 微分求一階條件 (FOC)。
  4. 施加對稱均衡條件 $b_1 = \beta(v_1)$。
  5. 求解常微分方程得到 $\beta(v)$。
