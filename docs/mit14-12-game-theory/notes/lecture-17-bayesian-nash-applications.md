## 本講主問題

本講透過具體的經濟應用實例來練習求解貝式納許均衡（Bayesian Nash Equilibrium, BNE）。主要探討兩個場景：未知需求下的庫諾競爭（Cournot competition），以及不完全資訊下的連續型拍賣模型（Auctions），特別是第一價格密封拍賣的均衡行為與求解技巧。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| Cournot with Unknown Demand | 庫諾模型中引入不完全資訊，一家廠商知道需求截距，另一家不知。 | 完整展示離散型 BNE 的設定、策略空間與求解聯立條件。 |
| First-Price Sealed-Bid Auction | 第一價格密封拍賣。最高出價者得標，並支付自己的出價。 | 介紹規則，並以雙人均勻分配為例求解線性對稱 BNE。 |
| Second-Price Sealed-Bid Auction | 第二價格密封拍賣。最高出價者得標，支付第二高出價。 | 介紹規則與直覺，說明真實出價的誘因。 |
| Symmetric Linear Equilibrium | 在拍賣求解中，猜測所有玩家採用相同的線性出價策略（$b = \alpha v$），再透過最佳反應驗證並求解係數。 | 作為求解連續型 BNE 的標準技巧「猜測與檢驗（Guess and Check）」來介紹。 |
| Bid Shading | 在第一價格拍賣中，投標者會出價低於自己真實估值以賺取利潤。 | 強調得標機率與得標利潤之間的取捨。 |

## 重要細節

- 定義（解概念／賽局元素／經濟量）：
  - 估值（Valuation）：若以該價格得標，效用剛好為 0 的最高願付價格（break-even）。
- 定理敘述與證明思路：
  - 第一價格拍賣線性對稱均衡求解：猜測 $B_2(v) = \alpha v$，最大化期望效用，得出最佳反應 $b_1 = v_1/2$，對照猜測確認 $\alpha = 1/2$。
- 賽局實例（矩陣、賽局樹、payoff 設定；標明「依口語重建」或 `待補`）：
  - 未知需求的 Cournot 競爭（依口語重建方程式）：需求為 $P = \theta - Q$。廠商 1 知道 $\theta$（高低各 1/2 機率），廠商 2 不知道。
  - 第一價格密封拍賣：兩人，估值服從獨立的 $U[0,1]$ 分配。
- 課堂 MobLab 遊戲：規則、結果、講者詮釋：
  - 第一價格拍賣：學生傾向過度出價（overbid，高於 Nash 均衡預測），因為太想贏。這反映了人們在實驗室中的真實偏好。
  - 第二價格拍賣：學生出價非常接近真實估值，效率較高。因為在第二價格拍賣中，出價不影響得標後的支付金額。
- 經濟應用場景：
  - 公債拍賣（Treasury auctions）、慈善拍賣、Google 廣告（Search ads & 3rd-party）、古董拍賣。拍賣能達到「價格發現」（Price discovery）的功能。
- 講者例子或直覺說明：
  - 為什麼 Google Ads 不定價而是用拍賣？因為不同廣告主對不同關鍵字的估值差距極大，拍賣能最大化收益並有效分配。
- 問答重點：
  - 學生問：在第一價格拍賣期望效用公式中，遇到平手怎麼辦？講者答：因為估值是連續分配，平手（出價完全相同）的機率為零，可忽略不計。
- 容易忽略的提醒：
  - 廠商策略的定義：即使只觀察到一種狀態，知情玩家的策略必須是「完整的應變計畫（complete contingent plan）」，包含所有可能類型的行動。

## 解概念與均衡分析

若本講有具體解概念或均衡分析（如 dominance、rationalizability、Nash、SPNE、BNE、PBE），記錄：

- 解概念名稱：Bayesian Nash Equilibrium (BNE)
- 形式化定義：每位玩家的每一個類型（Type）都在給定其他人的策略下，最大化自己的期望效用。
- 求解程序或演算法：
  1. 定義每個玩家的類型空間與策略函數。
  2. 寫出每個玩家每個類型的期望效用函數（需對未知狀態取期望值）。
  3. 對於連續型賽局，使用「猜測與檢驗（Guess and Check）」：猜測某種形式的對稱均衡（如 $b = \alpha v$），求出單一玩家給定猜測下的最佳反應，最後檢驗最佳反應是否符合原本猜測的形式以解出參數。
