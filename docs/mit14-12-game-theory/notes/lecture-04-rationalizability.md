## 本講主問題

- 上一講建立了「理性（rational）＝選擇對某個信念（belief）的最佳反應（best response）」，也證明了「一策略若非某信念的最佳反應，則被某混合策略嚴格優勢（strictly dominated）」的二分定理。本講要問：如果玩家不只自己理性，還「知道」對手理性、甚至「知道對手知道自己理性」……，這種**高階理性知識（higher order knowledge of rationality）**會如何進一步限縮可能被玩的策略？
- 為此本講把上述推理形式化成一個**演算法**：可理性化（rationalizability），程序名稱為**嚴格劣勢策略的反覆刪除（iterated elimination of strictly dominated strategies, IESDS）**。
- 引入「條件式嚴格優勢（strictly conditional dominance）」與「限制信念的最佳反應集合」兩個記號，把上週的二分定理推廣到任意受限的對手策略子集。
- 定義**可被優勢求解（dominant solvable）**：反覆刪除後每位玩家只剩唯一策略的賽局。
- 用三個例子演示：3×2 主例（可求解到唯一結果）、囚犯困境、一個 2×2 無法刪除任何策略的例子，最後以連續策略的**選美賽局（beauty contest / α-beauty contest）**收尾。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| 理性（rational） | 玩家 I 理性 ＝ 她所選策略是對「某個關於對手策略的信念」的最佳反應。對信念本身無任何限制，故單獨的理性是很弱的假設。 | 書稿「形式化與定義」重述，連結 [優勢](../03-dominance.md) 的二分定理 |
| 高階理性知識 | 一階：我知道對手理性；二階：我知道對手知道我理性；三階……。逐層加深，逐步限縮信念與可玩策略。 | 用文字＋逐步刪除表呈現 |
| 限制信念的最佳反應集合 $BR_i(S'_{-i})$ | 對「只在 $S'_{-i}$ 上放正機率的信念」為最佳反應的玩家 I 策略集合。 | 形式化段落，set-builder 記號 |
| 條件式嚴格優勢 $SCD_i(S'_{-i})$ | 在「對手只玩 $S'_{-i}$ 內策略」的條件下，被某混合策略嚴格優勢的策略集合。 | 形式化段落 |
| 二分定理（推廣版） | 對任意子集 $S'_{-i}$：$S_i = BR_i(S'_{-i}) \cup SCD_i(S'_{-i})$ 且兩集合交集為空（恰在其一）。$S'_{-i}$ 變小則 $BR$ 變小、$SCD$ 變大。 | 定理敘述＋單元素特例說明 |
| IESDS 演算法 | 遞迴定義：$S_i^0=S_i$；$S_i^{k+1}=BR_i(S_{-i}^k)=S_i \setminus SCD_i(S_{-i}^k)$；輸出 $S_i^\infty=\bigcap_k S_i^k$。 | 以流程與遞迴式呈現 |
| 可理性化策略 | 通過每一輪、永不被刪除的策略（$S_i^\infty$ 的元素）。 | 定義與命名區分（性質 vs 程序） |
| 可被優勢求解（dominant solvable） | 反覆刪除後每位玩家只剩一個策略、給出唯一預測的賽局。 | 主例與選美賽局各示一次 |
| 有限性收斂 | 有限賽局中集合每輪只縮不脹、無循環，必在有限步後靜止（某玩家連兩輪不變不夠，須全體都不變才算結束）。 | 常見誤解與小結 |
| 解題技巧 | 過程中用 $SCD$（漏刪可補、寧可多留），最後一輪用 $BR$ 檢查（避免留太多）。 | 常見誤解段落 |

## 重要細節

### 記號與慣例
- 全課從本講起一律用小寫 $u$ 表示效用，理解上包含期望效用（涉及信念、混合策略、自然行動時）。上週強調的大寫 $U$（期望效用，定義在 lottery 上）與 von Neumann–Morgenstern 效用（定義在 consequence 上）的概念區分仍成立，只是記號統一為小寫以減負擔。
- 策略型賽局寫成 $u:S_1\times\cdots\times S_N\to \mathbb{R}^N$，把策略組合 $s$ 映到 payoff 向量 $u(s)=(u_1(s),\dots,u_N(s))$。講者強調「整個賽局理論某種意義上就是研究這種函數」：$N$ 給玩家數、$S$ 給各人策略集、函數本身給所有 payoff。
- 玩家 I 的信念（belief）是對「所有其他玩家所選策略（部分組合 $s_{-i}$）」的信念。
- 標號慣例：下標＝玩家、上標＝輪數／消去階段（例如 $S_1^1$ 讀作「玩家一在第一輪後的策略集」）。

### 定義（形式化）
- **理性**：玩家 I 理性 $\iff$ 其策略是對某信念的最佳反應（等價：不玩被混合策略嚴格優勢的策略）。
- $BR_i(S'_{-i})=\{s_i\in S_i:\ s_i \text{ 是對某信念 }\beta_{-i}\text{ 的最佳反應，且 }\beta_{-i}(S'_{-i})=1\}$。即信念對 $S'_{-i}$ 以外的策略放零機率。
- $SCD_i(S'_{-i})=\{s_i\in S_i:\ s_i \text{ 在 } S'_{-i} \text{ 條件下被某混合策略 } \sigma_i \text{ 嚴格條件式優勢}\}$。
- **嚴格條件式優勢**定義：$u_i(\sigma_i, s_{-i}) > u_i(s_i, s_{-i})$ 對所有 $s_{-i}\in S'_{-i}$ 成立。優勢關係永遠是同一玩家自己策略之間的比較；信念才是對對手策略。

### 定理敘述與證明思路
- 上週定理的推廣：對任意 $S'_{-i}\subseteq S_{-i}$，$S_i = BR_i(S'_{-i})\cup SCD_i(S'_{-i})$（每策略至少屬其一），且 $BR_i(S'_{-i})\cap SCD_i(S'_{-i})=\varnothing$（至多屬其一）→ 恰屬其一。
- 單調性：$S'_{-i}$ 變小 ⇒ 可用信念變少 ⇒ $BR$ 變小；⇒ 要嚴格優勢的不等式變少、更易被優勢 ⇒ $SCD$ 變大。因聯集恆等於 $S_i$，一縮一脹。
- 單元素特例：若 $S'_{-i}=\{s_{-i}\}$，$BR_i$ 就是對該單一策略的最佳反應集；$SCD_i$ 就是「不是最佳反應」的策略集——單一對手策略下，非最佳反應 $\iff$ 存在其他策略嚴格較優 $\iff$ 條件式嚴格優勢。
- 收斂性（有限賽局）：策略有限、每輪只刪不增、無循環（不會刪掉又放回），故有限步後靜止。

### 賽局實例

#### 主例 3×2 bi-matrix（依口語重建，payoff 明確且一致）
玩家一選列 {T, M, B}，玩家二選欄 {L, R}；每格「玩家一, 玩家二」。

|  | L | R |
|---|---|---|
| **T** | 2, 0 | -1, 1 |
| **M** | 0, 10 | 0, 0 |
| **B** | -1, -6 | 2, 0 |

- 玩家一最佳反應：對 L 選 T（得 2）；對 R 選 B（得 2）。M 對 L、R 皆非最佳，且被 $\tfrac12 T+\tfrac12 B$ 嚴格優勢（對 L：$0.5>0$；對 R：$0.5>0$）→ M 永非最佳反應。
- 玩家二最佳反應：對 T 選 R（1>0）；對 M 選 L（10>0）；對 B 選 R（0>-6）。
- 逐步刪除：
  - 步驟 1（雙方理性）：$S_1'=\{T,B\}$、$S_2'=\{L,R\}$（玩家二兩策略皆為某純策略的最佳反應，無法刪）。
  - 步驟 2（各自知道對手理性，一階）：$S_1''=\{T,B\}$（不變）、$S_2''=\{R\}$（玩家一只玩 T/B，R 對兩者皆嚴格優於 L，對其混合亦然）。
  - 步驟 3（知道對手知道自己理性，二階）：$S_1'''=\{B\}$（玩家二只玩 R，B 是對 R 的最佳反應）、$S_2'''=\{R\}$。
  - 結果：唯一策略組合 (B, R)，兩格 payoff 皆被劃線（B 是對 R 的最佳反應、R 是對 B 的最佳反應）。此賽局**可被優勢求解**。
- 講者強調「crisscross（交叉）」結構：$S_1^{k+1}$ 由「玩家二在第 $k$ 輪的 $S_2^k$」決定，$S_2^{k+1}$ 由 $S_1^k$ 決定，每步都回看上一步。

#### 囚犯困境（Prisoner's Dilemma）
- 策略：Cooperate（與另一囚犯合作＝不出賣）、Defect。講者澄清 cooperate 指與「另一囚犯」合作而非與當局合作。
- 具體 payoff 數值本講**未給出**（`待補`）；只口述結論：D 對 C 嚴格優勢（對手 C 時 D 較好、對手 D 時 D 亦較好），故第一輪即刪 C 留 D，$S_i^1=\{D\}$。末端檢查 D 對 D 為最佳反應。單一策略即知已結束。

#### 2×2「無可刪」例（策略 T/B 對 L/R）
- 具體 payoff 數值本講**未給出**（`待補`）；講者僅口述最佳反應結構：
  - 玩家一：對 L 偏好 T；對 R 對 T 與 B 無異（皆得 0），兩者都要劃線。
  - 玩家二：對 T 偏好 L；對 B 對 L 與 R 無異，兩者都要劃線。
  - 檢查：T 是對 L（及 R）的最佳反應；B 是對 R 的最佳反應；L 是對 T、B 的最佳反應；R 是對 B 的最佳反應。故 $S_1^1=\{T,B\}$、$S_2^1=\{L,R\}$，無任何刪除。
- 教訓：可理性化有時給極強預測（主例、囚犯困境），有時完全無法刪除任何策略。強調有多個最佳反應時**兩者都要劃線**。

#### 選美賽局（α-beauty contest，連續策略）
- 設定：玩家 $i=1,\dots,N$，各選實數 $x_i\in[0,100]$（本講放寬為任意實數，非整數）。參數 $\alpha\in(0,1)$；第一天上課玩 $\alpha=2/3$。
- Payoff（二次損失 quadratic loss）：$u_i=-\left(x_i-\alpha\cdot\frac{x_1+\cdots+x_N}{N}\right)^2$。猜中 $\alpha\times$ 全班平均得最高 payoff 0，越遠 payoff 越負。
- 微妙處：自己的猜測 $x_i$ 也計入全班平均。
- 重寫：把 $x_i$ 提出，$u_i=-\big((1-\tfrac{\alpha}{N})x_i-\alpha\tfrac{N-1}{N}\bar{x}_{-i}\big)^2$，其中 $\bar{x}_{-i}$ 為「除我之外」的平均。
- 最佳反應：令括號為 0 解 $x_i$，得 $x_i=\dfrac{\alpha(N-1)}{N-\alpha}\,\bar{x}_{-i}$。記此係數為 $\alpha_N=\dfrac{\alpha(N-1)}{N-\alpha}$。
- 直覺：$\alpha_N<1$（分子減 1、分母減更小），故最佳猜測「嚴格小於」$\alpha\times$ 平均——因為自己的猜測會把全班平均往下拉。$N\to\infty$ 時 $\alpha_N\to\alpha$（個人對平均的影響變小）。
- IESDS：$S_i^0=[0,100]$；對手最高皆猜 100 時我想猜 $\alpha_N\cdot100$，最低皆猜 0 時我猜 0，故 $S_i^1=[0,100\alpha_N]$；一般 $S_i^k=[0,100\alpha_N^{\,k}]$。因 $\alpha_N<1$，極限 $S_i^\infty=\{0\}$。此賽局可被優勢求解，唯一可理性化策略為每人猜 **0**。
- 講者說此變體出現在作業（problem set），故課堂只快速帶過。

### 問答重點（學生提問）
- 「$S_{-i}$ 縮小時 $BR$ 一定嚴格縮小，還是可能維持不變？」→ 可能維持不變；演算法終會停止，某輪不再刪即完成；後續選美/2×2 例即示範不變的情形。
- 「$SCD$ 給單一策略時為何『其餘全部』都被嚴格優勢？」→ 單一對手策略下，條件式嚴格優勢等同「非最佳反應」，故非最佳反應者全落入 $SCD$。
- 「遞迴步驟為何寫 $BR_i(S_{-i}^k)$ 而不寫成 $S_i^k$ 內取？」→ 有限賽局中把定義域限成 $S_i^k$ 會得同答案；概念上寫成「對 $S_{-i}^k$ 的最佳反應」更貼近演算法精神；無限賽局需小心。
- 「演算法會不會把策略刪掉又放回（swap）？」→ 不會；集合每輪只縮不脹、無循環。
- 「有限策略型賽局是指玩家數有限還是每人策略數有限？」→ 兩者皆有限；實質重點是各 $S_i$ 有限（$N$ 通常也不取無限）。

### 容易忽略的提醒
- 單獨「理性」對信念無限制，是很弱的假設；限縮來自「知道對手理性」等高階知識。
- 理性不一定會刪掉策略（主例中玩家二在步驟 1 無可刪）。
- 演算法「結束」不等於「每人只剩一策略」；結束＝連續一輪無任何玩家被刪。只要有任一玩家被刪就須續行。
- 無限（連續策略）賽局本課會照樣套用這些概念求解（如選美賽局），只是不對無限情形做形式化保證（可能有病態不連續反例）。考試若要求算可理性化策略，不能以「非有限賽局」為由拒答。
- 有多個最佳反應時務必全部劃線。

## 解概念與均衡分析

- 解概念名稱：可理性化（rationalizability）；程序名稱：嚴格劣勢策略反覆刪除（IESDS）。
- 形式化定義：
  - 遞迴：$S_i^0=S_i$；$S_i^{k+1}=BR_i(S_{-i}^k)=S_i\setminus SCD_i(S_{-i}^k)$，其中 $S_{-i}^k=\prod_{j\ne i}S_j^k$。
  - 輸出：$S_i^\infty=\bigcap_{k=0}^{\infty}S_i^k$，其元素稱「玩家 I 的可理性化策略」。
- 求解程序：從全策略集開始逐輪刪除嚴格（條件式）劣勢策略，直到「所有玩家」策略集連兩輪不變。實作建議：過程用 $SCD$、末端用 $BR$ 驗證。
- 適用範圍與限制（依本講講者所述）：形式化保證限「有限」策略型賽局；連續策略賽局照樣套用但不做形式化保證。輸出可能是唯一（可被優勢求解），也可能完全無法刪除。
- 在解概念精煉鏈上的位置：介於[優勢](../03-dominance.md)與 [Nash 均衡](../05-nash-equilibrium.md)之間。可理性化＝反覆施行「不玩嚴格劣勢策略」＋高階理性知識；比單輪優勢更精煉，但（一般而言）仍比 Nash 均衡寬鬆——本講以「dominant solvable 時 (B,R) 兩格互為最佳反應」點到與均衡的呼應，正式的 Nash 均衡於下一講引入。
