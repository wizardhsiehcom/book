## 本講主問題

- 上一講的[可理性化](../04-rationalizability.md)（$S^\infty$）有時仍給不出銳利預測（如本講的波士頓賽局，四個策略組合全部可理性化）。本講引入賽局理論最核心的解概念——**Nash 均衡（Nash equilibrium）**——把預測再收緊一層。
- 核心對照：可理性化只要求玩家「對自己的玩法有一套能無限深入的解釋（信念）」，但那套解釋**未必正確**；Nash 均衡進一步要求解釋（信念）**也正確**，即精確描述對手實際怎麼玩。這就是「信念一致性（consistency of beliefs）」。
- 給出 Nash 均衡的三種等價表述：(1) 無人有嚴格獲利的單邊偏離（unilateral deviation）；(2) 不等式定義 $u_i(s_i^*,s_{-i}^*)\ge u_i(s_i',s_{-i}^*)$；(3) 每位玩家的策略互為對手策略的最佳反應。
- 建立解概念集合的包含關係：$DSE\subseteq NE\subseteq S^\infty\subseteq S$，並逐一給證明（DSE⊆NE 直接；NE⊆S^∞ 用歸納法）。
- 引入**混合策略 Nash 均衡（mixed strategy Nash equilibrium, MSNE）**，用捉迷藏（無純策略均衡）與波士頓賽局（純＋混合均衡並存）示範求解；以「無異（indifference）」為求解核心。
- 陳述 **Nash 存在定理**：每個有限策略型賽局至少有一個（混合策略）Nash 均衡；不證，僅點出用固定點定理。最後給 Nash 均衡三種詮釋（穩定慣例、自我執行協議、穩態）。

## 重要細節

### 記號與慣例（沿用前三講）
- 效用一律小寫 $u_i$（沿用[可理性化](../04-rationalizability.md)一章的統一慣例；涉及信念或混合策略時理解為期望效用）。
- 純策略 $s_i\in S_i$、策略組合 $s=(s_i,s_{-i})$、對手組合集 $S_{-i}$；信念 $\beta_{-i}$；最佳反應 $BR_i$。
- 混合策略用小寫 $\sigma_i$；「所有混合策略的集合」用大寫 $\Sigma_i$。講者明列四層記號：小寫 $s$＝純策略、大寫 $S$＝純策略集、小寫 $\sigma$＝混合策略、大寫 $\Sigma$＝混合策略集。
- 均衡加星號 $s^*=(s_1^*,\dots,s_N^*)$、$\sigma^*=(\sigma_1^*,\dots,\sigma_N^*)$。

### 可理性化回顧（本講開頭）
- 重述 IESDS 的階段解讀：$S^0$（無假設，全策略組合）→ $S^1$（理性：人人對某信念最佳反應）→ $S^2$（理性＋知道對手理性）→ …$S^K$（理性＋ $K-1$ 階理性知識）→ $S^\infty$（可理性化＝理性的共同知識，common knowledge of rationality）。
- 假設越強、預測越銳利；但 $S^\infty$ 有時仍不夠銳利，故需 Nash。
- $S^K$ 是乘積：$S^K=S_1^K\times\cdots\times S_N^K$。

### Nash 均衡定義（形式化）
- **是一個策略組合。** 講者強調考試常見錯誤：被要求給均衡時，學生給的是 payoff 而非策略組合；均衡永遠是「對玩家怎麼玩的預測／建議」，即策略組合。
- 定義：$s^*$ 是 Nash 均衡，若對每位玩家 $i$ 與其所有策略 $s_i'\in S_i$：
$$u_i(s_i^*,\,s_{-i}^*)\;\ge\; u_i(s_i',\,s_{-i}^*).$$
兩邊對手玩法固定為 $s_{-i}^*$（我控制不了對手），只比較我自己的策略。$s_i'=s_i^*$ 時自動成立。
- **等價一（最佳反應）**：$s_i^*$ 是對 $s_{-i}^*$ 的最佳反應。更形式地說是對「把機率 1 放在 $s_{-i}^*$ 的信念 $\beta_{-i}$」的最佳反應（$\beta_{-i}(s_{-i}^*)=1$）。定義裡看不到信念，是因為要求信念一致：信念＝實際玩法。
- **等價二（單邊偏離）**：無玩家有「嚴格獲利的單邊偏離」。偏離＝與均衡策略不同的策略；單邊＝只有我改。不等式兩邊對手玩法相同，唯一差別是我的策略。獲利以效用（期望效用）計，非金錢。可以有讓 payoff 相同的偏離，但不能有讓 payoff 嚴格較高的偏離。
- 人物與年份（講者口述，`待查` 精確出處）：以 John Nash 命名，經濟學諾貝爾獎；講者說他在 1948／1949 年寫了幾篇引入此概念的論文，原始論文「只有兩段長」。存在定理歸於 Nash 1948。

### 定理與證明思路
- **解概念集合包含鏈**：$DSE\subseteq NE\subseteq S^\infty\subseteq S$。三者都是「策略組合的集合」，皆為 $S$ 的子集。$S^\infty\subseteq S$ 是定義自明。
- **證 $DSE\subseteq NE$**（直接）：設 $s^*$ 是優勢策略均衡（DSE）。對任一玩家 $i$、任一偏離 $s_i'$：因 $s_i^*$ 是弱優勢策略，它弱優勢 $s_i'$，故 $u_i(s_i^*,s_{-i})\ge u_i(s_i',s_{-i})$ 對**所有** $s_{-i}$ 成立，特別對 $s_{-i}^*$ 成立 → $s_i^*$ 是對 $s_{-i}^*$ 的最佳反應。對所有玩家皆然 → $s^*$ 是 Nash 均衡。高層直覺：「若我的做法無論對手怎麼玩都最好，那麼在對手特定的玩法下當然也最好。」注意此證明**不保證** DSE 存在，只說「若有 DSE，它必是 NE」。
- **證 $NE\subseteq S^\infty$**（歸納法）：設 $s^*$ 是 Nash 均衡，要證 $s^*\in S^\infty$，等價於「對所有 $K$，$s^*\in S^K$」（即通過每一輪刪除、存活到底）。
  - 基底 $K=0$：$s^*\in S^0$（全策略組合），自明。
  - 歸納步：若 $s^*\in S^K$ 則 $s^*\in S^{K+1}$。**不能**逐玩家各自推理，因為「玩家 $i$ 哪些策略存活到 $K+1$ 輪」取決於「對手哪些策略存活到 $K$ 輪」，須同時推理。關鍵用的是歸納假設中「對手部分」：$s_{-i}^*\in S_{-i}^K$。而 $S_i^{K+1}$＝對「$S_{-i}^K$ 上某信念」的最佳反應集。由 Nash 定義，$s_i^*$ 是對 $s_{-i}^*$ 的最佳反應，且 $s_{-i}^*\in S_{-i}^K$（歸納假設），故 $s_i^*$ 是對「把機率全放 $s_{-i}^*$（落在 $S_{-i}^K$ 內）的信念」的最佳反應 → $s_i^*\in S_i^{K+1}$。對所有玩家皆然，歸納完成。
- **Nash 存在定理**：每個**有限**策略型賽局至少有一個混合策略 Nash 均衡（純策略為退化混合的特例）。講者說這是 Nash 1948 的關鍵結果，也是全課數學上最深的結果，用**固定點定理（fixed point theorem）**（有時歸於 Brouwer、有時 Kakutani，其一約 1944 年證出）。課程不證；講義（lecture notes）有建立在這些定理上的證明。「至少一個」——可能唯一、可能多個、甚至無限多，但至少一個。

### 賽局實例

#### 波士頓賽局（Boston game, BoS）——依口語重建
兩位朋友喜歡彼此作伴但運動偏好不同（Celtics vs. Red Sox）；同去比分開好，玩家一偏好 Celtics、玩家二偏好 Red Sox。講者說「這次用稍微不同的數字讓後面數學好算」。逐字稿後段明確口述「Celtics Red Sox … 2 1 1 2 0 0 0 0」，據此重建（每格「玩家一, 玩家二」）：

|  | C（Celtics） | R（Red Sox） |
|---|---|---|
| **C（Celtics）** | 2, 1 | 0, 0 |
| **R（Red Sox）** | 0, 0 | 1, 2 |

- 最佳反應（雙方對稱）：對手出 C 我出 C、對手出 R 我出 R。
- 可理性化：全部四個策略組合 {CC, CR, RC, RR} 都可理性化（$S^\infty$ 有四元素）→ 不夠銳利。
- 其中 RC（玩家一去 Red Sox、玩家二去 Celtics）可理性化卻「像是犯錯」：兩人都落單、各自單邊改都會更好。講者用此說明可理性化＝「有一套能無限深入的解釋」，但解釋可能不正確（我以為朋友也去 Red Sox，事實上他去了 Celtics）。
- 純策略 Nash 均衡：CC 與 RR。兩格皆兩數劃線。從 RR 想到 CC 不是單邊偏離（我只能改自己去哪）。
- DSE：**空集**。玩家一：C 不弱優勢 R（對手出 R 時 C 得 0 < R 得 1）；R 也不弱優勢 C（對手出 C 時 R 得 0 < C 得 2）→ 無弱優勢策略。故無 DSE。此例正好示範 $DSE\subsetneq NE\subsetneq S^\infty$：CC、RR 是 NE 非 DSE；CR、RC 可理性化非 NE。

#### 捉迷藏（Hide and Seek）——依口語重建
純粹利益衝突（zero-sum）。玩家一＝躲藏者（兩處 A、B），玩家二＝搜尋者。搜尋者猜中躲藏處則贏。也可講成罰球員 vs. 守門員、逃稅者 vs. 國稅局。重建（每格「玩家一, 玩家二」）：

|  | A | B |
|---|---|---|
| **A** | −1, 1 | 1, −1 |
| **B** | 1, −1 | −1, 1 |

- 最佳反應：玩家一（躲）想與對手**不同**格：對手 A→我 B、對手 B→我 A。玩家二（搜）想與對手**相同**格：對手 A→A、對手 B→B。
- 劃線後**無任何格兩數皆劃線** → **無純策略 Nash 均衡**。直覺：同格時躲者想偏離、異格時搜者想偏離，輸家永遠想改。
- 混合：直覺答案是 50/50（不要可預測）。講者說世界盃罰球資料大致 50/50（右腳球員略有左偏等細節）。

### 混合策略 Nash 均衡（定義與求解）
- **定義**：混合策略組合 $\sigma^*=(\sigma_1^*,\dots,\sigma_N^*)$ 是 Nash 均衡，若對每位玩家 $i$：$u_i(\sigma_i^*,\sigma_{-i}^*)\ge u_i(\sigma_i',\sigma_{-i}^*)$ 對所有 $\sigma_i'\in\Sigma_i$ 成立。
- **只需檢查純策略偏離**：上式等價於只要求對所有純策略偏離 $s_i'\in S_i$ 成立。直覺：對純策略混合所得 payoff 是各純策略偏離 payoff 的加權平均；若每個純偏離都弱較差，任何混合也弱較差。**關鍵**：均衡策略「可以」是混合的，但檢查偏離時只需看純策略。
- **無異原理**：只有在對所混合的純策略「無異」時才願意混合（嚴格偏好其一就不會給另一者正機率）。且**更強**：只會在「純最佳反應」之間混合——無異是**必要非充分**。講者的反例：A、B、C payoff 各 1、1、2，雖 A、B 無異但不會混 A、B（C 嚴格較好）；若再加 D＝2，才會在 C、D 間混。
- **捉迷藏驗證半半**：設玩家一混 50/50。加一列「½A+½B」代表玩家一用此混合策略時的 payoff：把原矩陣上下兩列各乘 ½ 相加，得該列為 (0,0,0,0)。結論：玩家二出 A 得 0、出 B 得 0 → 無異 → A、B 皆純最佳反應 → 玩家二混半半是最佳反應。對稱可得另一方向。故半半是均衡（講者口語以打勾略過完整對稱步驟，並提醒作業不可略）。
- **波士頓混合均衡**：找 $0<P_1<1$、$0<P_2<1$，玩家一以 $P_1$ 出 C、玩家二以 $P_2$ 出 C。均衡要各自無異。
  - **交叉關鍵**：使玩家二無異的是 $P_1$（玩家一的混合決定玩家二的信念），使玩家一無異的是 $P_2$。
  - 玩家二無異：出 C 得 $P_1\cdot1+(1-P_1)\cdot0=P_1$；出 R 得 $(1-P_1)\cdot2$。令相等 $P_1=2(1-P_1)\Rightarrow P_1=2/3$。
  - 玩家一無異：出 C 得 $2P_2$；出 R 得 $(1-P_2)$。令相等 $2P_2=1-P_2\Rightarrow P_2=1/3$。
  - 解得 $P_1=2/3$、$P_2=1/3$。
  - **常見誤解澄清**：不是「玩家一常去 Celtics 因為他更愛 Celtics」。真正原因是**對手（玩家二）較不愛 Celtics**——玩家一的混合必須讓玩家二無異，而玩家二不太愛 Celtics，只有相信玩家一很常去 Celtics 才願意去。反向同理：玩家二常去 Red Sox 是為了讓偏好 Celtics 的玩家一願意去 Red Sox。
  - **未解的哲學問題**：我既然無異，為何要恰好用讓對手無異的機率去混合？講者留待後續，並給兩種常見說法：(1) 混合策略常是「重複賽局／動態過程」的故事——你不想有「總是射左」的名聲讓對手更新；(2) 「混合」未必是字面隨機化，而是依對手觀察不到的訊號（天氣決定穿著、罰球員臨場的體感方向）做決定，在外人看來像隨機。

### 問答重點（學生互動）
- 波士頓賽局 RC「哪裡怪」→ 學生答：兩人都在輸，各自改就更好。
- 找波士頓純均衡 → 學生答 CC、RR。
- 證 NE⊆S^∞ 的技巧 → 學生提「反證法」與「歸納法」；講者採歸納（因 $S^\infty$ 本就是歸納定義）。
- 捉迷藏怎麼玩 → 學生答「混合策略」、「半半」。
- 為何願意混合 → 學生先答「A 在某些情況好、B 在某些情況好」，講者引導到「給定對手玩法必須無異」；另一學生問「對手是否也得混」→ 講者說相關但先擱置，稍後回到「正因對手在混，我才無異」。

### 容易忽略的提醒
- 均衡是**策略組合**，不是 payoff。
- 定義裡「看不到信念」是因為要求信念一致（$\beta_{-i}$ 對準真實 $s_{-i}^*$）。
- 檢查混合均衡偏離時只需檢查純策略偏離；但均衡策略本身可以是混合的。
- 無異是願意混合的**必要非充分**條件；還要所混合者皆為純最佳反應。
- 使對手無異的是「我的」混合機率，不是我自己的（易誤以為打錯字）。
- 純策略 Nash 均衡是混合策略 Nash 均衡的退化特例；「至少一個」存在定理涵蓋純與混合。
- 講者坦言：Nash 均衡的正當性未必比可理性化更有說服力（假設更強、預測更銳利，但是否可信不明顯）；考試仍須照解。

## 解概念與均衡分析

- 解概念名稱：Nash 均衡（Nash equilibrium）；混合推廣為混合策略 Nash 均衡（MSNE）。
- 形式化定義（三等價）：
  1. $u_i(s_i^*,s_{-i}^*)\ge u_i(s_i',s_{-i}^*)\ \forall i,\ \forall s_i'\in S_i$。
  2. $s_i^*\in BR_i(s_{-i}^*)$（對信念 $\beta_{-i}(s_{-i}^*)=1$ 的最佳反應）$\forall i$。
  3. 無玩家有嚴格獲利的單邊偏離。
  - 混合版：把 $s$ 換 $\sigma$、$S_i$ 換 $\Sigma_i$；檢查偏離只需純策略。
- 求解程序：
  - **純策略**：劃線法——對雙方所有最佳反應劃線，找兩數皆劃線的格。
  - **混合策略**：設未知混合機率，寫「無異條件」（使**對手**無異的方程），解機率；記得使對手無異的是我方機率。
  - 只在純最佳反應間混合（無異為必要非充分）。
- 適用範圍與限制（依本講講者所述）：存在定理僅對**有限**策略型賽局保證至少一個混合均衡，用固定點定理（Brouwer／Kakutani），課程不證。無異法求混合均衡在本講以 2×2 示範。
- 在解概念精煉鏈上的位置：$DSE\subseteq NE\subseteq S^\infty\subseteq S$。Nash 比[可理性化](../04-rationalizability.md)更苛（多要求信念正確／一致），比[優勢策略均衡](../03-dominance.md)寬。前接可理性化（dominant solvable 時「兩格互為最佳反應」已預告 Nash），後續延伸到 SPNE、BNE、PBE（[不完全競爭](../06-imperfect-competition.md)先做 Nash 的經濟應用）。
