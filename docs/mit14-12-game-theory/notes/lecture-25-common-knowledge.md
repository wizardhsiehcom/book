## 本講主問題

本講旨在形式化地定義「共同知識 (Common Knowledge)」。課程中經常假定「賽局本身是共同知識」，但過去未曾精確定義。本講透過狀態空間 (state space) 與資訊分割 (information partition) 的模型，定義何謂「知道」一件事，並區分「交互知識 (Mutual Knowledge)」與「共同知識」。最後透過「帽子謎題 (Hat puzzle)」具體展示將交互知識轉化為共同知識如何根本性地改變局中人的推論與推論行為。

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| 狀態空間 (State space) $\Omega$ | 描述世界所有可能狀態的集合，通常只包含與問題相關的維度。 | 定義與舉例（擲骰子） |
| 資訊分割 (Information partition) $K_i$ | 玩家 $i$ 觀察到的資訊，將狀態空間劃分為多個「細胞 (cells)」。對應廣泛形賽局的資訊集合。 | 定義並連結廣泛形賽局 |
| 事件 (Event) $E$ | 狀態空間的子集 $E \subseteq \Omega$。玩家對「事件」形成知識。 | 定義 |
| 知識 (Knowledge) | 在狀態 $\omega$ 下，若包含 $\omega$ 的細胞 $K_i(\omega)$ 是 $E$ 的子集，則玩家 $i$ 「知道」事件 $E$。 | 定義與舉例說明 |
| 交互知識 (Mutual Knowledge) $K(E)$ | 所有玩家都知道事件 $E$ 的狀態集合：$K(E) = \bigcap_i K_i(E)$。 | 定義與推廣至高階 |
| 共同知識 (Common Knowledge) $CK(E)$ | 所有人知道 $E$，所有人知道所有人知道 $E$，且無限延伸。 | 數學定義 |
| 帽子謎題 (The Hat Puzzle) | 說明互為交互知識（甚至二階交互知識）的事件被公開宣佈為共同知識後，如何改變系統的動態推論。 | 作為核心範例詳細拆解分析 |

## 重要細節

- 定義（解概念／賽局元素／經濟量）：定義了 $K_i(E) = \{\omega \in \Omega \mid K_i(\omega) \subseteq E\}$，表示玩家 $i$ 知道事件 $E$ 的所有狀態集合。
- 定理敘述與證明思路：證明 $K_i(E) \subseteq E$，即玩家只能知道「真實發生」的事。證明思路直接套用定義：若 $\omega \in K_i(E)$，則 $K_i(\omega) \subseteq E$；又因為 $\omega \in K_i(\omega)$，故 $\omega \in E$。
- 賽局實例（矩陣、賽局樹、payoff 設定；標明「依口語重建」或 `待補`）：
  - 擲骰子：狀態空間為 $\{1, 2, 3, 4, 5, 6\}$。舉例玩家 1 的分割為 $\{1, 2\}, \{3\}, \{4, 6\}, \{5\}$，說明不同狀態下對事件「點數不大於 4」是否具備知識。（依口語重建）
  - 帽子謎題：三位學生，每人戴紅或白帽。以三維座標 $(0/1, 0/1, 0/1)$ 表示狀態空間的立方體頂點。教師宣佈「至少有一頂紅帽」，將「至少一頂紅帽」的事件從二階交互知識變成了共同知識。（依口語重建）
- 課堂 MobLab 遊戲：規則、結果、講者詮釋：無。
- 經濟應用場景：無特定經濟應用，主要為邏輯與認識論在賽局理論的基礎建立。
- 講者例子或直覺說明：透過三維座標（立方體）畫出每個玩家的資訊分割（平行的線或面），生動展示「切分資訊集合」的概念。
- 問答重點：有學生問如果一直問下去是否會循環，講者提到其他版本的謎題確實會出現循環。學生問新資訊是否會讓人「忘記」原本知道的事，講者釐清不會。
- 容易忽略的提醒：交互知識（即使是高階）並不等於共同知識。帽子謎題中，「至少有一頂紅帽」在 $(1,1,1)$ 的狀態下是大家早就知道的，但公開宣佈後卻改變了後續的回答，正是因為改變了高階知識的結構。

## 解概念與均衡分析

- 解概念名稱：共同知識 (Common Knowledge)
- 形式化定義：$CK(E) = \bigcap_{n=1}^{\infty} K^n(E)$，其中 $K^1(E) = K(E)$，$K^n(E) = K(K^{n-1}(E))$。
- 求解程序或演算法：透過檢驗 $K_i(\omega) \subseteq E$ 來推導 $K_i(E)$，進一步求交集。在廣泛形或狀態圖上，則是透過「刪除狀態來切分資訊集合」。
- 適用範圍與限制（本講講者實際說明者）：任何形式化知識的討論。
- 在解概念精煉鏈上的位置與前後關聯：提供「賽局結構是共同知識」的嚴謹數學基礎，是所有賽局理論模型的根本假設。
