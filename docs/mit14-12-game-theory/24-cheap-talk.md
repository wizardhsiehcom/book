# 第24章：廉價對話 (Cheap Talk)

### 導讀

在此之前，我們探討了「訊號傳遞（Signaling）」，了解到擁有私有資訊的參與者，可以透過付出成本的行動（如接受教育、定價）來向另一方傳遞資訊。行動之所以能傳遞資訊，關鍵在於它對不同類型的人有不同的成本（差額成本）。

然而在真實世界中，許多資訊的傳遞是透過語言、文字等「無成本（costless）」的溝通方式進行的。當雙方利益完全一致時，這類溝通自然沒有問題；但如果雙方存在利益衝突（例如顧問與決策者對於政策的偏好不同），語言是否還能傳遞可信的資訊？本章將介紹著名的 Crawford-Sobel (1982) 模型，探討在無成本的「廉價對話（Cheap Talk）」中，如何在利益衝突下達成部分資訊揭露的均衡。

### 核心內容

#### 1. 廉價對話與訊號傳遞的差異

在訊號傳遞賽局中，傳遞資訊的行動（如教育年數）會直接影響參與者的報酬（payoff-relevant）。而在廉價對話中，訊息或言辭本身是沒有成本的，也不會直接進入參與者的效用函數（payoff-irrelevant）。

例如，求職面試時告訴雇主「我很優秀」是一句不用花錢的廉價對話。在嚴重的利益衝突下，雇主不會相信這句話，因為無論好壞員工都會這麼說。但本章要證明的是，在某些利益衝突較溫和的場景中，廉價對話依然能傳達有價值的資訊。

#### 2. Crawford-Sobel 模型設定

我們考慮一個由傳送者（Sender，如顧問）與接收者（Receiver，如決策者）組成的賽局：
- **狀態與類型**：系統存在一個未知狀態 $T$，且 $T$ 均勻分配於區間 $[0, 1]$。只有 Sender 能觀察到 $T$ 的真實數值。
- **訊息**：Sender 觀察到 $T$ 後，選擇發送一個訊息 $M$ 給 Receiver。發送任何訊息都沒有成本。
- **決策**：Receiver 觀察到訊息 $M$（但不知道 $T$），然後選擇一個行動 $Y \in \mathbb{R}$。

雙方的偏好可以用二次損失函數（Quadratic loss function）表示：
- **接收者 (Receiver) 的效用**：$U_R = -(Y - T)^2$
  這表示 Receiver 希望自己的行動 $Y$ 完全等於真實狀態 $T$。
- **傳送者 (Sender) 的效用**：$U_S = -(Y - (T + \beta))^2$
  這表示 Sender 希望 Receiver 的行動等於 $T + \beta$。其中 $\beta > 0$ 稱為**偏誤（Bias）**，代表雙方利益的衝突程度。

兩人對於「狀態越大，決策數值應越高」的方向是一致的（例如：經濟越差越需要降息；軍事衝突風險越高，越需要增加國防支出），但 Sender 總是希望決策比 Receiver 的理想值稍微高出 $\beta$（例如：政策顧問永遠比總統更傾向於增加支出）。

### 均衡分析

我們使用完美貝氏均衡（Perfect Bayesian Equilibrium, PBE）來尋找這個賽局的解。

#### 1. 無溝通均衡 (Babbling Equilibrium)

最極端的情況是，Sender 拒絕透露任何資訊，這稱為「喋喋不休（Babbling）」或無溝通均衡：
- **Sender 的策略**：無論狀態 $T$ 是多少，永遠發送相同的訊息 $M^*$。
- **Receiver 的策略**：既然訊息不包含任何資訊，Receiver 維持原本的先驗信念（Prior belief），認為 $T \sim U[0,1]$。因此 Receiver 選擇最大化期望效用的行動，即期望狀態值 $Y = 1/2$（區間中點）。
- **偏離誘因**：如果 Sender 傳送其他任何訊息，Receiver 都可以將其視為與 $M^*$ 無異的噪音，並同樣選擇 $Y = 1/2$。此時 Sender 無法透過改變訊息來改變 Receiver 的行為，因此沒有偏離的動機。這個均衡在任何 $\beta$ 下都永遠存在。

#### 2. 完全溝通的不可能性

我們能否有一個「完全揭露資訊」的均衡（即 Sender 總是誠實說出 $M = T$）？
假設這是一個均衡，Receiver 收到訊息 $M$ 後會相信狀態就是 $M$，因此選擇 $Y = M$。
但站在 Sender 的立場，如果他看到真實狀態為 $T$，他最想要的決策是 $Y = T + \beta$。於是，他會有極大的動機說謊，發送訊息 $M = T + \beta$ 以誘使 Receiver 做出他最喜歡的決策。因為 Sender 有誇大狀態的動機，Receiver 知道這一點後就不會完全相信 Sender，導致完全溝通崩潰。

#### 3. 分區均衡 (Partitional Equilibria)

雖然無法完全溝通，但可以進行「部分溝通」。這透過**分區均衡**來實現：Sender 將狀態空間 $[0,1]$ 劃分為若干個區間，只告訴 Receiver 狀態落在哪一個區間內（例如說「低」或「高」）。

以 $K=2$（兩個區間）為例：
假設我們以 $T_1$ 為分界點，將狀態分為「低 ($L$)：$T < T_1$」和「高 ($H$)：$T > T_1$」。
- 當 Receiver 收到 $L$ 時，他認為狀態均勻分佈在 $[0, T_1]$，因此最佳回應是區間的中點：$Y_L = T_1 / 2$。
- 當 Receiver 收到 $H$ 時，他認為狀態均勻分佈在 $[T_1, 1]$，因此最佳回應是：$Y_H = (T_1 + 1) / 2$。

要讓這個策略成為均衡，關鍵在於 Sender 的**誘因相容**。當狀態剛好等於閾值 $T_1$ 時，Sender 必須對發送 $L$ 和 $H$ 感到無異（Indifferent）。
換句話說，在 $T=T_1$ 時，Sender 的理想決策是 $T_1 + \beta$。為了讓他對 $Y_L$ 與 $Y_H$ 帶來的效用相同，二次損失函數的對稱性要求 $T_1 + \beta$ 必須剛好位於 $Y_L$ 與 $Y_H$ 的正中央：
$$T_1 + \beta = \frac{Y_L + Y_H}{2}$$
代入 $Y_L$ 與 $Y_H$：
$$T_1 + \beta = \frac{T_1/2 + (T_1 + 1)/2}{2} = \frac{T_1 + 1/2}{2}$$
$$2T_1 + 2\beta = T_1 + 1/2$$
$$T_1 = 1/2 - 2\beta$$

為了讓這個分界點在 $[0,1]$ 內，我們需要 $T_1 > 0$，因此推導出 $\beta < 1/4$。
這表示：只有當雙方的利益分歧 $\beta$ 夠小（小於 $1/4$）時，才有可能建立這種「高/低」兩區間的溝通均衡。

值得注意的是，這兩個區間的長度並不對稱。右邊（高）區間的長度是 $1 - T_1 = 1/2 + 2\beta$，左邊（低）區間的長度是 $T_1 = 1/2 - 2\beta$。右邊區間比左邊大了 $4\beta$。為什麼會這樣？這是為了抵銷 Sender 想要誇大狀態的偏誤。當 Sender 宣稱狀態很「高」時，因為高區間非常寬，Receiver 的更新信念會比較模糊，只會選擇這個寬大區間的中點，這樣的模糊性使得處於閾值的 Sender 不會想輕易越界謊報為「高」。

### 賽局實例與推廣：K 個區間

Crawford 與 Sobel 將這個邏輯推廣到了 $K$ 個區間的均衡。他們證明了：
1. **區間長度遞增**：在 $K$ 個區間的均衡中，相鄰兩個區間（由左至右）的長度是不斷遞增的，且後一個區間永遠比前一個區間長 $4\beta$。
2. **存在的條件**：存在 $K$ 區間均衡的充分必要條件是：
   $$\beta < \frac{1}{2K(K-1)}$$

這個結果帶來了幾個重要的經濟意涵：
1. **偏誤越小，溝通越精確**：隨著 $\beta$ 的縮小，方程式允許更大的 $K$ 值存在。當雙方利益越一致，Sender 能將狀態切割得越細（更多的 $K$），傳達越多細節。當 $\beta \to 0$ 時，幾乎能達到完全溝通。
2. **偏誤過大，無法溝通**：當 $\beta \ge 1/4$ 時，沒有任何 $K \ge 2$ 的均衡能成立。此時唯一的均衡是無溝通均衡，無論 Sender 說什麼，Receiver 都不會採信。
3. **資訊越豐富，雙方越有利**：給定一個 $\beta$，可能同時存在多個均衡（例如當 $\beta$ 很小時，可能同時存在 $K=1, 2, 3$ 的均衡）。在所有可能的均衡中，擁有最多區間（最高 $K$）的均衡，對 Receiver 與 Sender 在事前（Ex-ante）來說都是效用最高的。更多的資訊能讓決策更貼近狀態，這不只對決策者有利，對有偏誤但仍希望決策隨狀態變動的顧問來說也是有利的。

### 小結

Cheap Talk 理論提供了一個極具洞察力的視角：即使在沒有任何實質成本（不需要燒錢發送訊號）、且雙方存在利益衝突的情況下，只要分歧在一定範圍內，人們仍能透過「模糊但有意義」的分區方式進行實質的資訊交流。我們或許無法說出精確無誤的真相，但仍能誠實地讓對方知道目前的狀況是屬於「好」、「普通」還是「糟糕」，並以此增進雙方的福利。
