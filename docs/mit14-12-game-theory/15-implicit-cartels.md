# 第 15 章：隱含卡特爾與合謀 (Implicit Cartels)

## 15.1 導讀

上一章我們介紹了重複賽局與無名氏定理（Folk Theorem），證明了在無限期互動且耐心足夠的情況下，玩家可以透過未來的懲罰威脅來維持當期的合作。本章將把這個理論應用到經濟學中非常經典的主題：卡特爾（Cartels）與合謀。

卡特爾的典型例子是 OPEC（石油輸出國組織），他們透過協調各國的石油產量來維持高油價。其他的例子包含早期的維他命製造商卡特爾，或者是美國醫學會（AMA）對醫學院招生人數的限制（藉此限制醫生供給，維持醫師高薪）。

然而，維持卡特爾面臨一個根本的挑戰：**誘因不相容（Incentive to Deviate）**。當市場因為總產量減少而維持高價時，每個個別廠商都有強烈的動機去「偷偷增產」。由於增產導致價格下跌的外部性由全體承擔，但增產的營收卻是自己獨享，因此背叛合謀非常具有誘惑力。

如果是在一個國家內，廠商之間明文簽訂限制產量的合約通常是違反反托拉斯法（Antitrust laws）的；如果在國際上（如 OPEC），國家之間簽約缺乏有強制力的國際法庭來執行。因此，我們必須依賴**「隱含卡特爾」（Implicit Cartels）**——沒有明文合約，也沒有暴力威脅（例如毒品卡特爾劃分地盤那樣），完全依靠賽局內部的自我強制（Self-enforcing）機制來運作。本章將透過無限期重複的古諾（Cournot）競爭模型，分析如何建立次賽局完美納許均衡（SPNE）來維持這種隱含的合謀。

## 15.2 無限期古諾競爭模型與靜態基準

假設有 $N$ 家廠商，在 $t = 0, 1, 2, \dots$ 的無限期中進行 Cournot 數量競爭。
為了簡化，假設每家廠商的邊際成本均為常數 $C < 1$。
市場的反需求函數為 $P(Q) = \max\{1 - Q, 0\}$，其中 $Q = \sum_{i=1}^N q_i$ 為市場總產量。
廠商使用相同的貼現因子 $\delta$ 來計算未來的平均貼現利潤。我們並且假設「完美監控」（Perfect Monitoring），即廠商在每一期期末都能精確觀察到過去所有人的實際產量。

### 單次賽局的納許均衡 (Nash Equilibrium)

如果我們只看單次賽局，這就是標準的 Cournot 競爭。我們回顧其唯一的 Nash 均衡產量與利潤：
- 每個廠商的產量：$q^{NE} = \frac{1-C}{N+1}$
- 市場價格：$P^{NE} = 1 - N q^{NE} = C + \frac{1-C}{N+1}$ （邊際成本加上溢價 Markup）
- 每個廠商的利潤：$\pi^{NE} = \left(\frac{1-C}{N+1}\right)^2$

### 極大化總利潤的合謀產量 (Monopoly Quantity)

為了極大化所有廠商的總利潤，他們應該像一個獨占者（Monopolist）一樣行動。獨占的最適總產量為 $Q^M = \frac{1-C}{2}$。
若廠商均分這個產量，每家廠商的合謀產量為：
$$q^M = \frac{1-C}{2N}$$

顯然，$q^M < q^{NE}$。合謀產量帶來了更高的市場價格與更高的整體利潤。但這並非單次賽局的均衡，因為在其他人都生產 $q^M$ 時，單一廠商單方面增加產量會獲得更高的當期利潤。

## 15.3 自動機表示法與納許回復 (Nash Reversion)

為了維持 $q^M$，我們必須引入懲罰機制作為威脅。最簡單的懲罰策略是「納許回復」（Nash Reversion，或稱嚴厲觸發 Grim Trigger）：
- 正常情況下，大家都生產 $q^M$。
- 只要有任何一家廠商被發現偏離了 $q^M$，未來所有期所有人都永遠改為生產 $q^{NE}$。

為了分析這個策略是否構成 SPNE，我們引入**自動機表示法（Automaton Representation）**。這將重複賽局的複雜歷史簡化為一組「狀態」（States）與「轉移規則」（Transitions）。

對於 Nash Reversion 策略，我們可以定義兩個狀態：
1. **合謀狀態 (M)**：大家生產 $q^M$。若本期所有人都生產 $q^M$，下期保持在 M 狀態；若有任何人偏離，下期進入 NE 狀態。
2. **懲罰狀態 (NE)**：大家生產 $q^{NE}$。無論發生什麼事，下期永遠留在 NE 狀態（即此為吸收態）。

### 價值函數與均衡條件

為了方便計算，定義兩個函數：
- $F(q)$：當所有人都生產 $q$ 時，每家廠商的單期利潤。
- $G(q)$：當其他人都生產 $q$ 時，某廠商採取最佳單次偏離（Best Response）所能獲得的最大單期利潤。

我們為每個狀態計算「價值函數」$V$。這裡的價值函數指的是：假設所有人嚴格遵守該狀態指定的策略，從今以後的平均貼現利潤：
- $V_M = F(q^M)$
- $V_{NE} = F(q^{NE})$

接著利用**單次偏離原則（One-shot Deviation Principle）**，確保在每個狀態下，沒有人想進行單次偏離：
在 M 狀態下，遵守策略的報酬必須大於等於偏離的報酬：
$$V_M \geq (1-\delta) G(q^M) + \delta V_{NE}$$

將 $V_M$ 與 $V_{NE}$ 代入並移項，我們可以解出貼現因子的閾值：
$$\delta \geq \frac{G(q^M) - F(q^M)}{G(q^M) - F(q^{NE})} = \frac{\text{Gain}}{\text{Gain} + \text{Loss}}$$

其中：
- **Gain (G)** = $G(q^M) - F(q^M)$：今天偏離合謀所能多賺的短期利潤。
- **Loss (L)** = $F(q^M) - F(q^{NE})$：明天以後永遠遭受懲罰所喪失的長期利潤。

當廠商的耐心 $\delta$ 夠大時（大於該閾值），合謀就能自我維持。從公式中可以看出：
1. 如果懲罰造成的損失 $L$ 很大，閾值就會降低，廠商不需要那麼有耐心也能維持合作。
2. 如果偏離的短期獲利 $Gain$ 很大，閾值就會上升，合作更難維持。
3. **廠商數量 $N$ 的影響**：當參與的廠商數量 $N$ 增加時，每個廠商在合謀中的市佔率變小，導致偏離（偷偷增產）的相對誘惑變大。計算結果顯示閾值會隨著 $N$ 上升，這解釋了為何廠商數目眾多的卡特爾往往難以穩定存在，極易瓦解。

## 15.4 蘿蔔與棍子策略 (Carrot and Stick)

Nash Reversion 的懲罰是永久且無法挽回的，這在現實中並不常見。現實歷史上的價格戰（例如 1980 年代沙烏地阿拉伯為了懲罰其他超產國而大量增產，導致油價崩盤）通常只維持一段時間，懲罰過後大家會設法恢復合謀。這種有限期的懲罰被稱為「蘿蔔與棍子」（Carrot and Stick）策略。

我們可以設計一個具有兩個產量水準 $q^C$（低產量，蘿蔔）與 $q^S$（高產量，棍子）的自動機，其中 $q^C < q^S$：
1. **蘿蔔狀態 (C)**：大家生產 $q^C$。如果所有人都生產 $q^C$，下期維持 C 狀態；否則下期進入 S 狀態。
2. **棍子狀態 (S)**：大家生產 $q^S$（作為對偏離者的共同懲罰）。**如果所有人都乖乖執行了懲罰產量 $q^S$**，下期就會回到 C 狀態！若有人連懲罰都不願配合，下期則繼續留在 S 狀態。

### 價值函數方程組

這是一個非常巧妙的設計。即使偏離後的路徑看起來很複雜（例如偏離後會進入 S，然後回 C，再永遠在 C），自動機表示法可以將其轉換為簡單的聯立方程式。我們只需考慮「今天」與「明天及以後的狀態」：
- $V_C = (1-\delta)F(q^C) + \delta V_C$
- $V_S = (1-\delta)F(q^S) + \delta V_C$

（注意 $V_S$ 的第二項是 $\delta V_C$，因為只要遵守棍子狀態的懲罰，明天就會回到蘿蔔狀態。）
解聯立方程式非常容易，$V_C = F(q^C)$，而 $V_S$ 則反映了一期高產量的低利潤加上未來永遠合謀的高利潤。透過這種方式，不需要展開無限期的序列，也能精確計算出策略的長期報酬。

### 均衡條件檢驗

對於這兩個狀態，我們都需要確保最佳偏離無利可圖：
1. **在 C 狀態：** $V_C \geq (1-\delta)G(q^C) + \delta V_S$
   （偏離蘿蔔會導致多賺今天，但明天進入棍子狀態受罰）
2. **在 S 狀態：** $V_S \geq (1-\delta)G(q^S) + \delta V_S$
   （偏離棍子會導致明天「繼續」留在棍子狀態受罰）

只要廠商夠有耐心，並且 $q^S$ 的設計能讓「不配合懲罰」的短期利益比不上「乖乖受罰一期以回到合謀」的長期利益，這種有限期懲罰的卡特爾就能完美地自我維持。

## 15.5 小結

本章將無限期重複賽局的理論應用於卡特爾定價，展示了即使缺乏合約或法律保護，追求自身利益的寡占廠商也能透過「隱含」的未來懲罰威脅來維持高價合謀。

**自動機表示法（Automaton Representation）**為我們提供了一個強大且優雅的工具。它將複雜的無限期策略與歷史化約為簡單的「狀態轉移」與「價值聯立方程式」，並搭配「單次偏離原則」大幅簡化了均衡的檢驗。這套分析框架不僅解釋了 OPEC 為什麼能維持油價與偶發的價格戰，也從理論上證明了：卡特爾的穩定性高度依賴於廠商的耐心，且當廠商數量增加時會變得極端脆弱。
