# 第 12 章：有限重複賽局 (Finitely Repeated Games)

## 12.1 導讀

在先前的章節中，我們大多假設賽局只進行一次（One-shot game）。但在現實生活中，許多互動是持續且重複發生的。你與同一個人玩賽局、觀察結果，然後再次進行相同的賽局。本章將探討當一個賽局被有限次數重複時，結果會發生什麼改變。我們將回答一個核心問題：**重複的互動加上對過去行為的可觀察性，是否能促成單次賽局中無法達成的合作？** 透過分析有限重複賽局，我們將引入依賴歷史的獎懲機制，並發現階段賽局（Stage game）均衡的數量將決定重複賽局是否能發展出多元的互動模式。

## 12.2 核心內容：重複賽局的框架

有限重複賽局是多階段賽局（Multi-stage games）的一個特例。在每個階段中，所有玩家同時行動，且玩的都是同一個**階段賽局（Stage game）**。

### 12.2.1 模型的構成要素

一個有限重複賽局包含以下元素：
1. **階段賽局 $G$**：這是一個常態形式賽局（Strategic form game），包含玩家的行動集合與收益函數。
2. **時間結構**：賽局在期數 $t = 0, 1, 2, \dots, T$ 進行，共重複 $T+1$ 次。$T$ 稱為賽局的視界（Horizon），且為有限值。我們將重複賽局記為 $G(T)$。若 $T=0$，則 $G(0)$ 就是階段賽局本身。
3. **資訊結構（可觀察性）**：在每一期開始前，所有玩家都能完美觀察到過去所有期數中每一位玩家的行動。
4. **收益（Payoffs）**：玩家在重複賽局中的總收益通常以**平均收益**計算。若玩家 $i$ 在每期的收益為 $u_i(a_t)$，則其在 $G(T)$ 中的總收益為：
   $$ \frac{1}{T+1} \sum_{t=0}^{T} u_i(a_t) $$
   使用平均收益的好處是，重複賽局的收益單位能與階段賽局保持一致，便於比較。

### 12.2.2 歷史與策略

重複賽局中最重要的概念是**歷史（History）**。第 $t$ 期的歷史 $h_t$ 包含了從期數 $0$ 到 $t-1$ 所有玩家的行動組合。
- 期數 0 的歷史為空集合 $\emptyset$。
- 期數 1 的歷史 $h_1$ 是期數 0 的行動組合 $a_0$。
- 期數 $T$ 的歷史 $h_T$ 是序列 $(a_0, a_1, \dots, a_{T-1})$。

玩家的**策略（Strategy）**是一個完整的應變計畫（Complete contingent plan），必須指定在**每一個**可能的歷史下應採取的行動。重複性加上可觀察性，賦予了玩家設定**獎勵與懲罰（Rewards and Punishments）**的可能——即「如果今天你合作，我明天就合作；如果你今天背叛，我明天也背叛」。

## 12.3 形式化與定理：唯一納許均衡的侷限

若階段賽局只有一個納許均衡，重複賽局的結果會如何？我們以經典的囚徒困境（Prisoner's Dilemma）為例。

### 12.3.1 囚徒困境的有限重複

假設囚徒困境的收益矩陣如下（行動為 C: 合作, D: 背叛）：

| | C | D |
|---|---|---|
| **C** | 2, 2 | -1, 3 |
| **D** | 3, -1 | 0, 0 |

在此單次賽局中，(D, D) 是唯一的納許均衡。如果我們將此賽局重複兩次（期數 0 與期數 1），我們能找到支持 (C, C) 的子賽局完美納許均衡（SPNE）嗎？

根據向後歸納法（Backward Induction），在最後一期（期數 1），不管期數 0 發生了什麼，玩家面對的都是一個不會影響未來的單次賽局。因此，他們只能選擇最大化當期收益的行動，即唯一的納許均衡 (D, D)。

既然期數 1 的行動固定為 (D, D)，且**完全不受期數 0 的歷史影響**，那麼在期數 0 時，玩家就沒有任何未來獎懲的誘因。因此，在期數 0，玩家同樣只能選擇最大化當期收益的行動，依然是 (D, D)。

### 12.3.2 唯一納許均衡定理

上述邏輯可以推廣為一個普遍的定理：

**定理**：令 $G$ 為一個常態形式賽局。如果 $G$ 有**唯一**的納許均衡 $a^*$，那麼對於任何有限視界 $T$，重複賽局 $G(T)$ 具有唯一的子賽局完美納許均衡（SPNE）。在此均衡中，無論發生什麼歷史，玩家在每一期都選擇階段賽局的均衡行動 $a^*$。

**直覺證明**：
1. 在最後一期 $T$，因為沒有未來，玩家的行為不會改變未來的歷史。為使策略在該子賽局構成均衡，必須且只能玩階段賽局的納許均衡 $a^*$。
2. 來到第 $T-1$ 期，玩家知道無論自己做什麼，第 $T$ 期大家都只會玩 $a^*$。既然未來的行為獨立於當前歷史，未來就不存在獎懲機制。因此，玩家在第 $T-1$ 期也只能最大化當期收益，選擇 $a^*$。
3. 依此類推回推至第 0 期，每一期必定選擇 $a^*$。

這意味著，如果階段賽局只有一個均衡，有限次數的重複是**無法**帶來合作或改變互動結果的。

## 12.4 賽局實例與應用：多重均衡與獵鹿賽局

如果我們希望在有限重複賽局中看到歷史相依的獎懲機制，階段賽局就必須擁有**多重納許均衡**。

### 12.4.1 獵鹿賽局 (Stag Hunt)

考慮盧梭（Rousseau）提出的獵鹿賽局。兩名獵人可以合作獵鹿（S），也可以單獨獵兔（H）。
- 獵鹿成功收益高，但必須兩人合作，若一人獨自獵鹿則會失敗且一無所獲。
- 獵兔收益低，但可單獨完成。

| | S | H |
|---|---|---|
| **S** | 2, 2 | 0, 1 |
| **H** | 1, 0 | 1, 1 |

此賽局有兩個純策略納許均衡：
1. **(S, S)**：好均衡（收益為 2, 2）
2. **(H, H)**：壞均衡（收益為 1, 1）

### 12.4.2 利用多重均衡建立獎勵與懲罰

假設我們將獵鹿賽局重複兩次（期數 0 與 1）。因為最後一期（期數 1）必須玩階段賽局的納許均衡，但在這裡我們有**兩個**選擇：(S, S) 或是 (H, H)。

我們能否建構一個 SPNE，使得在期數 0 時，玩家願意選擇非均衡的行動（例如 (H, S)）？
答案是可以的。我們可以利用 (S, S) 作為乖乖聽話的**獎勵**，並將 (H, H) 作為偏離軌道的**懲罰**。

考慮以下建構：
- **期數 0 行動**：玩家 1 選擇 H，玩家 2 選擇 S。這本身不是納許均衡。
- **期數 1 策略（依賴歷史）**：
  - 如果期數 0 順利出現了 (H, S)，則期數 1 大家玩 (S, S) 作為獎勵。
  - 如果任何玩家單方面偏離了 (H, S)（例如玩家 1 偏離為 S 導致歷史變成 (S, S)，或玩家 2 偏離為 H 導致歷史變成 (H, H)），則期數 1 大家玩 (H, H) 作為懲罰。

**利用單次偏離原則（One-shot Deviation Principle）檢驗：**
在期數 0，玩家 1 應該乖乖選 H 嗎？
- 若乖乖選 H，當期得 1 分，下期得 2 分（因為會進入 (S,S)），平均收益為 $(1+2)/2 = 1.5$。
- 若偏離選 S，當期得 2 分（因為會變成 (S,S)），但下期受懲罰得 1 分（因為會進入 (H,H)），平均收益為 $(2+1)/2 = 1.5$。
因為偏離並沒有帶來嚴格更高的收益，所以玩家 1 沒有偏離的誘因。

同理檢驗玩家 2：
- 若乖乖選 S，當期得 0 分，下期得 2 分，平均為 $1.0$。
- 若偏離選 H，當期得 1 分，下期受懲罰得 1 分，平均為 $1.0$。
玩家 2 亦沒有偏離的誘因。

透過這個例子我們看到，**階段賽局中的多重均衡是建立有限重複賽局中獎懲機制的必要條件**。

## 12.5 常見誤解

1. **以為只要重複玩就能產生合作：** 許多人憑直覺認為在重複互動中，玩家為了未來的利益會傾向於當下合作。然而理論證明，在有限視界且唯一均衡的情境（如有限次囚徒困境）下，理性的結果依然是從頭到尾不合作。
2. **混淆策略與結果（Outcome）：** 在重複賽局中，「結果」是一條實際發生的行動路徑（例如：每一期都背叛）。但「策略」是一個龐大的函數，必須交代在所有可能發生的歷史下玩家的反應（例如：即使我們計劃每期背叛，我的策略仍必須說明如果上期有人意外合作了，我這期該怎麼做）。

## 12.6 小結

有限重複賽局展示了動態互動中誘因的本質。只要人們能夠觀察到過去的歷史，就能將未來的行動與過去的行為掛鉤。然而，由於有限重複賽局在最後一期沒有未來，必須退化為階段賽局的納許均衡。因此，如果階段賽局只有單一納許均衡，獎懲機制便無法由後往前建立；但若存在多重納許均衡，我們就能將好均衡與壞均衡作為獎勵與懲罰，從而支撐起豐富的互動行為。在後續章節中，我們將看到當賽局被**無限次**重複時，這種對多重均衡的依賴將會被打破。
