# 導讀：為什麼現在要懂 CoPoS

2026 年，AI 加速器的封裝面積需求已經逼近 12 吋晶圓的物理極限——CoWoS 的矽中介板再大，也塞不進客戶想要的 HBM 數量。台積電的答案是把「圓的」變成「方的」：**CoPoS（Chip-on-Panel-on-Substrate）**，用 310 × 310 mm 的矩形面板取代圓形晶圓作為封裝載體，可用面積放大五倍以上，材料利用率從不到 70% 提升到 90% 以上。

CoPoS 試產線已於 2026 年年中完成，規劃 2027 年試產、2028 下半年起量產。它不只是 CoWoS 的放大版，而是牽動玻璃基板、面板級 RDL、設備與材料供應鏈的一次典範轉移。理解 CoPoS，就是理解 2030 年前 AI 硬體封裝的主戰場。

## 這本筆記的目標

1. 補齊理解 CoPoS 所需的前提知識：先進封裝基礎、CoWoS 架構、面板級封裝（FOPLP）脈絡。
2. 建立 CoPoS 的完整心智模型：為什麼從圓到方、為什麼用玻璃、面板級製程的真正難點。
3. 掌握 2026 年的最新進度、時程與供應鏈布局。
4. 看清 CoPoS 與 CoWoS、SoW-X、競爭陣營（Samsung、Intel、Absolics）的相對位置。
