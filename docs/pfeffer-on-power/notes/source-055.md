## 核心內容

- Pfeffer 在軍事領導語境中把 power 比喻成 potential energy，influence 則是權力被使用時的狀態。
- 如果好人選擇不參與權力遊戲，就不能抱怨權力落到不好的人手上。
- 軍隊也有評鑑、金字塔和競爭；好人若不經營能見度與關係，會從金字塔掉下來。
- Rule 1 包括移除冒牌者症候群、過度追求被喜歡和錯誤理解 authenticity。
- Rule 2 是 break rules, not laws；社會改革、醫療創新與軍事創新常需要挑戰慣例。
- Rule 3 透過聲音、身體、眼神和空間傳遞權力。
- Rule 4 的品牌可以來自專業媒體或社群，例如軍事部落格。
- Rule 5 的 networking 是 generosity，也包括幫部屬找到機會。
- 權力有代價：能見度、努力、嫉妒和成為目標；需要 personal board 和 truth-tellers。
