## 實際主題

Jon Levy 討論如何透過 Influencer Dinners 建立高信任關係網絡。他把社會科學原理應用到活動設計：共同做事、匿名互動、降低地位階層、創造新奇與驚喜、篩選參與者，讓高影響力人士在沒有明顯索取壓力的場域中建立關係。

## 核心概念

- 關係、信任與網絡會改變人生機會。
- 免費活動不代表沒有商業價值；關係本身比短期收費更值錢。
- IKEA effect：人們會更重視自己參與製作的東西。
- Vulnerability loop：先交換小型脆弱訊號，信任才會加速形成。
- 高影響力人士仍是人，也會孤單，也會擔心別人只想要錢、時間、access 或 status。
- Novelty 和 surprise 會提高注意力、探索意願與記憶。
- 匿名晚餐降低身份階層，讓人先作為人互動，再知道彼此成就。
- Consistency 比一開始就有頂級名單更重要。
