## 實際主題

Dafina Toncheva 談從 Bulgaria 到 Harvard、Microsoft、Stanford、Venrock 與 USVP 的路徑，重點是大量要求與申請、策略性選擇需要自己的團隊、面對性別偏見時自我倡議，以及在 venture 中管理 LP 敘事與支持網絡。

## 核心概念

- You don't get what you don't ask for：從 Bulgaria 申請美國學校本身就是要求與風險承擔。
- 選工作不只看名氣，要看哪個地方最需要你、最願意支持你。
- 轉型中的組織風險較高，但也可能提供領導機會。
- 面對不可行動的偏見回饋，不必接受，也不必過度個人化；用工作結果取得位置。
- 在 venture 裡，管理 LP 關係與敘事和投資本身同樣重要。
- 自己的成功與福祉是自己的責任；要成為自己的 cheerleader。
- 厚臉皮不是冷漠，而是慎選誰有權影響你對自己的看法。

## 重要案例

| 案例 | 用途 |
|---|---|
| 申請 50 所美國學校 | 說明大量要求與風險不對稱。 |
| Venrock 兩年 associate program | 說明用 mentorship 換取密集學習。 |
| 選擇 USVP 而非其他名氣更穩的 offer | 說明加入最需要你的團隊。 |
| CEO 要求她退出 board | 說明面對性別、年齡與外貌偏見時自我倡議。 |
| 成為 board chair | 說明用承擔 tedious work 取得實際領導位置。 |
| 管理 LP narrative | 說明成果需透過敘事被資金提供者理解。 |
