## 核心內容

- Jason Calacanis 的開場 blueprint 主張：在組織裡選一個具體、稀缺、別人不想做但有價值的技能，迅速學會，整理最佳實務，訓練他人，成為 fixer。
- Pfeffer 訪談重申七條規則：人們常把理想世界和現實世界混在一起，因此拒絕學習權力。
- Rule 1：停止自我貶低、過度道歉和過度追求被喜歡；他人更關心你是否能把事情做成。
- Rule 2：打破規則常帶來差異化；商業模式創新和社會運動都需要挑戰慣例。
- Rule 3：身體語言與非語言線索會改變他人對權力的判斷。
- Rule 4：品牌需要長期一致；所謂一夕成名常是二十年以上累積。
- Rule 5：多數人低估社交關係，過度投資技術或產品本身；networking 要提供價值。
- Rule 6：權力要被使用，否則不能帶來資源與結果。
- 訪談也以 Laura Esserman、Amir Rubin、Trump 等案例說明權力可用於好事，也可被危險地使用。
