# 附錄 A：術語表

| 英文 | 暫譯 | 本書用法 |
|---|---|---|
| power | 權力 | 讓他人、資源、制度或結果朝某方向移動的能力。 |
| influence | 影響力 | 不一定依賴正式職位，但能改變他人判斷或行動的能力。 |
| personal brand | 個人品牌 | 別人記得你、理解你、把某類機會和你連在一起的方式。 |
| networking | 經營網絡 | 建立可交換資訊、幫助與機會的人際基礎設施。 |
| authenticity | 真誠／本真 | 待補；本書會區分親密關係與工作場域中的不同意義。 |
| weak ties | 弱連結 | 非親密但能帶來非重複資訊與機會的人際關係。 |
| hierarchy | 階層 | 組織中正式或非正式的上下位置與決策權分布。 |
| status | 地位 | 他人給予的尊重、敬重與社會排序。 |
| warmth | 溫暖 | 他人是否覺得你友善、親和、無威脅。 |
| competence | 能力 | 他人是否覺得你能完成事情、做出判斷、承擔角色。 |
| self-enhancement | 自我美化 | 人傾向把自己看得比平均更好，並喜歡與自己相似的人。 |
| acting with power | 演出權力 | 透過身體、聲音、空間與角色感傳遞權威與可被注意性。 |
| playing high / low | 演高位／低位 | 依任務需要切換支配、承擔、退讓或讓位的角色姿態。 |
| strategic communication | 策略溝通 | 以聽眾、情境、結構與目標設計訊息，而不只是把內容說完。 |
| spontaneous speaking | 即興表達 | 在問題、會議、回饋、道歉或閒聊中即時組織語言的能力。 |
| job control | 工作控制感 | 員工對工作時間、方法、節奏與決策的自主程度。 |
| social pollution | 社會污染 | 組織把工作造成的健康與生活成本外部化給個人、家庭與社會。 |
| job lock | 工作鎖定 | 因健康保險、收入或其他制度依賴而難以離開不適合工作的狀態。 |
| amplifying the work | 放大成果 | 主動讓團隊貢獻、影響與需求被組織看見。 |
| top of mind | 優先被想起 | 當機會出現時，別人第一時間想到你或你的團隊。 |
| virtual team | 虛擬團隊 | 沒有正式編制在同一團隊，但因共同敘事和目標而投入的人群。 |
| investable person | 值得投資的人 | 讓他人相信你具備能力、關係、使命與執行力，值得投入資源。 |
| sponsor | 贊助者 | 願意把自己的信用、關係或權力押一部分在你身上的資深人士。 |
| shamelessness | 厚臉皮／不怕尷尬 | 把不好意思從行動決策中移開，仍能主動要求機會與資源。 |
| reputation structures | 聲譽結構 | 協會、商會、董事會、媒體或專業平台等能替個人能力提供外部驗證的場域。 |
| perception becomes reality | 感知成為現實 | 他人對你能力與成功機率的相信，會影響他們是否提供資源，進而改變實際結果。 |
| imposter syndrome | 冒牌者症候群 | 即使有客觀成就，仍害怕自己不配、會被看穿的心理狀態。 |
| conflict capacity | 衝突承受力 | 能把分歧留在桌上處理，而不逃避、爆炸或把人際關係燒掉的能力。 |
| steel man | 鋼人化 | 先用最強版本理解並表述對方論點，再回應分歧。 |
| contextual authenticity | 情境化真誠 | 真誠不是固定表演，而是在不同場域中使用適合該情境的自己。 |
| credibility asset | 信用資產 | 長期誠實、準確與可驗證行為累積出的可信度。 |
| leader as vessel | 領導者作為載體 | 外界透過領導者理解組織、運動、產品或使命。 |
| social proof | 社會證據 | 他人透過你與哪些人、平台、成果連在一起，判斷你是否值得信任。 |
| show up bigger | 以更大姿態出場 | 早期個人或組織用結果、能量、敘事與社會證據呈現超出現有規模的可信力量。 |
| only in the room | 房間裡的唯一者 | 在場域中因性別、族裔、年齡或背景而成為少數甚至唯一代表的人。 |
| platform leverage | 平台槓桿 | 把會議、校園、刊物、活動或組織位置轉成能見度、關係與聲譽資產。 |
| narrative management | 敘事管理 | 主動整理並重複傳達成果、方向與角色，讓關鍵聽眾理解你的價值。 |
| support network | 支持網絡 | 由主管、同儕、贊助者與盟友構成，能支撐自我價值與職涯推進的人際基礎。 |
| centrality | 中心性 | 位在多方關係、資訊或資源流動的交會點，讓他人需要透過你連接彼此。 |
| nonverbal cues | 非語言線索 | 姿勢、眼神、距離、聲音、表情與手勢等會被他人解讀的身體訊號。 |
| perceived power | 被感知的權力 | 他人根據外在訊號相信你擁有的權力，不一定等於實際控制資源的能力。 |
| actual power cues | 實際權力線索 | 較能反映誰真正控制互動的線索，如成功打斷、較長說話時間與較大音量。 |
| visual dominance ratio | 視覺支配比例 | 說話時看著對方、聽對方說話時移開目光的比例；可被讀成權力訊號。 |
| field design | 場域設計 | 主動設計活動、媒介、榜單、訪談或空間，讓特定關係和權力互動發生。 |
| vulnerability loop | 脆弱迴圈 | 一方釋出小型脆弱訊號，另一方承接並回應脆弱，進而加速信任形成。 |
| IKEA effect | IKEA 效應 | 人們會更重視自己參與製作或投入勞動的事物。 |
| curation | 策展／篩選 | 有意識地選擇誰進入場域、如何互動、什麼行為被獎勵或排除。 |
| host power | 主持者權力 | 創造房間、安排介紹、分配注意力與讓結果發生的人所得到的不成比例信用。 |
| demonstrable growth | 可見成長 | 每次產品、活動或媒介都比上一次明顯更好，讓外界看見進步軌跡。 |
| information node | 資訊節點 | 位在資訊流入與轉譯位置的人，能把他人分散的觀察變成可用判斷。 |
| disciplined opportunism | 有紀律的機會主義 | 主動讀取新機會，但只追求能和核心能力、組織利益與長期路徑互相強化的機會。 |
| career portfolio | 職涯投資組合 | 由能力、關係、外部品牌、內部盟友、選項與生活條件組成的風險分散結構。 |
| inherited person | 被繼承的人 | 原 sponsor 離開後，被新主管接手但尚未取得新信任的成員。 |
| job crafting | 角色塑造 | 主動把工作內容調整到更貼近自己的強項、興趣和組織需要。 |
| cognitive load | 認知負荷 | 持續監控、解讀或擔心某些情境所消耗的注意力與心智資源。 |
| fiduciary duty | 受託責任 | 投資或管理資產時對受益人負有的審慎責任；本書也用來討論偏見如何扭曲資本配置。 |
| movement narrative | 運動敘事 | 讓分散個案被理解成共同問題，並讓支持者知道自己正在參與什麼改變的故事結構。 |
| human sustainability | 人的永續 | 把員工健康、控制感、壓力、訓練與工作品質視為需要被測量和維護的組織責任。 |
| high commitment work system | 高承諾工作系統 | 由慎選、訓練、授權、資訊透明、團隊與成果分享等互相支撐的工作實務組成。 |
| preemptory apology | 預先道歉 | 在發言或行動前先降低自己位置與內容價值的語言習慣。 |
| personal board of directors | 個人董事會 | 由教練、朋友、導師或同儕組成，能給回饋、追蹤行動並要求你實踐權力技能的支持結構。 |
| flywheel effect | 飛輪效應 | 成功帶來更多資源、人才與信任，進一步增加成功機率的自我強化過程。 |
| doing power project | 權力實作專案 | 將權力規則用在具體情境中，實驗如何改變品牌、網絡、資源或職涯位置的練習。 |
| cofounder risk | 共同創辦人風險 | 共同創辦初期補足能力，後期卻可能形成權力鬥爭、責任模糊或外部資本介入的風險。 |
| behavioral interviewing | 行為式面談 | 以候選人過去實際經驗、行動、結果和學習判斷能力，而非只問假設題。 |
| intelligent failure | 智慧型失敗 | 從失敗中萃取可遷移的判斷、流程與行動原則，而不只是把失敗當成羞恥。 |
| high beta arena | 高波動場域 | 可能帶來巨大影響與巨大損失的權力場，如政治競選、公共職務或早期新創。 |
| reputation system | 聲譽系統 | 透過評價、回饋、紀錄或社群規則，讓陌生人能判斷彼此可信度的制度設計。 |
| show, don't tell | 展示勝於宣稱 | 用可觀察的產品、行動或結果取得信任，而不只用投影片或口頭承諾說服。 |
| fixer | 解題者 | 在組織中因能處理特定棘手問題而被記住、被尋求、被賦予機會的人。 |
| relationship flow | 關係流 | 資訊、信任、介紹、資源與機會在人際網絡中移動的路徑。 |
| dependency management | 依賴管理 | 在跨部門工作中主動管理他人承諾、風險、查證與替代路徑的能力。 |
| organizational triage | 組織分診 | 判斷哪些問題需要立即處理、哪些可以等待，以及哪些風險最值得優先測試。 |
| collective power | 集體權力 | 透過工會、運動、社群或聯盟把分散個體組織成能改變制度的力量。 |
| quid pro quo | 對價交換 | 為取得資源、位置或保護而承諾回報；本書特別用來提醒靠近權力時的風險。 |
| autonomy price | 自主性代價 | 高權力位置常以時間、隱私、行程控制和家庭生活為交換成本。 |
