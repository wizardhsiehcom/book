# 導讀：CS234 強化學習 (Reinforcement Learning)

歡迎來到 Stanford CS234: Reinforcement Learning (Winter 2026/Spring 2024)。

## 課程概述
強化學習 (Reinforcement Learning, RL) 是讓人工智慧系統學習做出良好決策的重要典範。它的應用範圍非常廣泛，涵蓋機器人技術、遊戲、消費者行為建模以及醫療保健等領域。

本知識庫將帶你逐講精讀 Emma Brunskill 教授的 CS234 課程逐字稿，涵蓋從基礎的 MDP、Q-learning，一直到進階的 Offline RL、探索機制 (Exploration)、多智能體遊戲 (Multi-Agent Game Playing) 與 RLHF 價值對齊 (Value Alignment)。

## 學習地圖

1. **基礎概念與規劃 (Foundations & Planning)**
   - Lecture 1: Introduction to RL
   - Lecture 2: Tabular MDP Planning
2. **免模型評估與控制 (Model-Free Evaluation & Control)**
   - Lecture 3: Policy Evaluation
   - Lecture 4: Q-learning and Function Approximation
3. **策略搜尋 (Policy Search)**
   - Lecture 5: Policy Search 1
   - Lecture 6: Policy Search 2
   - Lecture 7: Policy Search 3
4. **離線強化學習 (Offline RL)**
   - Lecture 8: Offline RL 1
   - Lecture 10: Offline RL 3
5. **探索機制 (Exploration)**
   - Lecture 11: Exploration 1
   - Lecture 12: Exploration 2
   - Lecture 13: Exploration 3
6. **進階與前沿主題 (Advanced Topics)**
   - Lecture 9: Guest Lecture on DPO (Rafael Rafailov, Archit Sharma, Eric Mitchell)
   - Lecture 14: Multi-Agent Game Playing
   - Lecture 15: Guest Lecture (Emma Brunskill & Dan Webber)
   - Lecture 16: Value Alignment
