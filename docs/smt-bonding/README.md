# 熱風・熱壓・熱板　電子接合完整指南

## 本書目標

電子製造中，**焊接與接合**是品質的核心。本書以三種最常見的加熱工藝為主軸：

| 工藝 | 加熱方式 | 典型應用 |
|------|---------|---------|
| 熱風回流 | 強制對流 | SMT 量產主線 |
| 熱壓接合 | 傳導（刀頭接觸） | FPC、ACF、顯示器模組 |
| 熱板傳導 | 傳導（板面接觸） | 打樣、維修 |
