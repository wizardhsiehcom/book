## 導讀

大自然經過數百萬年的演化，創造出各式各樣適應不同環境的「相機」——動物的眼睛。從最簡單的光斑、針孔眼，到複雜的複眼與折射透鏡，動物的眼睛為了生存，在「方向解析度 (directional resolution)」與「感光度 (sensitivity)」之間取得了不同的平衡。本章將帶領讀者一窺這些生物學上的奇妙設計，並探討它們如何啟發未來的計算相機技術。

## 核心內容

動物眼睛的演化，可視為一道由簡至繁的「成像階梯」：從只能判斷明暗方向的眼點，逐步加上凹陷、針孔、透鏡，直到廣角的複眼。每一階都在「方向解析度」與「感光度」之間做出不同取捨。

```mermaid
flowchart LR
    A["眼點<br/>Eye Spot<br/>只知方向"] --> B["眼凹<br/>Eye Pit<br/>方向性更佳"]
    B --> C["針孔眼<br/>Pinhole Eye<br/>能成像、進光少"]
    C --> D["透鏡眼<br/>Lens Eye<br/>聚光＋高解析"]
    D --> E["複眼<br/>Compound Eye<br/>廣視野、對運動敏感"]
```

1. **基礎功能與取捨**：
   眼睛的基本功能是比較來自不同方向的光強。演化上主要針對兩個指標進行最佳化：方向解析度與感光度。例如深海動物需要極高感光度，而需要精確捕食的動物則需要高解析度。

2. **感光細胞與視紫紅質 (Rhodopsin)**：
   視紫紅質是感光的核心蛋白質，由視黃醛 (retinal) 與視蛋白 (opsin) 組成。當光子擊中時會改變形狀並觸發神經訊號。某些鳥類甚至擁有能感知磁場的 Cryptochromes，而昆蟲則能透過感光細胞的特定排列來感知光的偏振 (polarization)，以此導航。

3. **從眼點 (Eye Spot) 到針孔眼 (Pin-hole Eye)**：
   - **Eye Spot**: 單一感光細胞加上色素遮蔽，只能感知光的方向，如眼蟲 (euglena)。
   - **Eye Pit**: 凹陷的結構提供更好的方向性，類似蛇類的紅外線感測。
   - **Pin-hole Eye**: 如同針孔相機，雖然能成像但進光量小且受繞射限制。加入角膜與透明體液後，即為現代眼球的雛形。

4. **透鏡演化與像差校正**：
   - **球面透鏡與 GRIN Lens**：動物眼中的晶狀體多為球面，這會帶來球面像差。但大自然發展出了漸變折射率透鏡 (GRIN Lens)，中心折射率高、邊緣低，使光線在透鏡內平滑彎曲，不僅消除像差也縮短了焦距。
   - **多重透鏡**：如橈足類 (Pontella) 擁有多個透鏡來逐步折射光線，減少最終像差，這啟發了現代相機的多鏡片設計。

5. **瞳孔與動態範圍**：
   貓等夜行性動物演化出直條狀的瞳孔 (slit pupils)，相較於人類的圓形瞳孔，能更快速且大範圍地調節進光量。夜行動物同時擁有反射層 (Tapetum lucidum)，讓光線兩次穿過視網膜以提升感光度。

6. **複眼 (Compound Eyes)**：
   由大量小管 (ommatidia) 組成，每個小管只負責極小的視角。雖然解析度極差（受限於繞射），但視野廣闊且對運動極為敏感。

## 計算攝影的啟發

動物眼睛的設計不斷啟發著現代科技：
- **微透鏡陣列**：如同複眼結構，被應用於 CCD/CMOS 感測器與光場相機中。
- **液態透鏡 (Fluid lens)**：模仿動物肌肉擠壓晶狀體改變焦距的機制。
- **多焦點與 Foveon 感測器**：模仿動物視網膜不同深度的感光層設計，或利用色散特性設計多焦點系統。
- **NASA 的多解析度相機**：模仿具有視覺帶 (visual streak) 或中心凹 (fovea) 的動物眼睛，動態分配運算資源。

## 小結

本章透過生物學的視角，重新審視了「成像系統」的可能性。大自然早已實作了許多計算攝影中嘗試解決的問題，無論是光學上的 GRIN Lens，或是感測器層級的偏振與多光譜感測。這些「動物之眼」不僅是演化的奇蹟，更是未來相機設計的豐富靈感庫。
