# 前言

## 本書緣起

本書是 MIT Media Lab 課程 **MAS.531 / MAS.131 _Computational Camera and Photography_（Fall 2009）** 的繁體中文精讀筆記。原課程由 **Ramesh Raskar** 教授主講，內容橫跨現代光學、計算照明、光場、光譜成像到編碼成像，是計算攝影領域早期最具代表性的系統性課程之一。

計算攝影的核心主張，是把相機從「被動記錄光線的暗箱」重新理解為**光學硬體與演算法的共同設計（co-design）**：在照明、光圈、快門、感測器上主動加入「編碼」，再以軟體「解碼」，就能取得傳統 2D 相機無法捕捉的高維度資訊——4D 光場、深度、光譜，甚至看穿遮蔽物與轉角。本書希望把這一整套思路，以中文讀者容易入門的方式重新組織與保存。

## 資料來源

本書所有內容整理自 **MIT OpenCourseWare（OCW）** 公開釋出的課程材料，主要包括：

- **課堂錄影逐字稿**：作為各章敘述與講者例子的主要依據。
- **講義 PDF（Study Materials）**：作為系統架構、公式與規格數字的核對來源。
- **指定閱讀清單（Readings）**：各項技術的原始論文出處。

書中出現的規格數字（如光場相機解析度、論文年份、器材參數）皆可回溯至[參考資料](references.md)頁所列的一手來源；凡與課堂口述有出入或需要換算之處，均在正文以註記說明，不靜默竄改。完整的課程頁、syllabus、影片鏡像與逐講對照表，同樣收錄於參考資料頁。

## 與原課程的對應與取捨

本書章節依課程 Lecture 順序編排，但並非每一講都獨立成章，取捨原則說明如下：

- **Lecture 7（紅外線成像、斷層掃描與 3D 技術，兩場客座）於 OCW 未釋出任何講義、影片或音訊材料**。由於缺乏一手依據，若強行補寫必然流於臆造，故本書不為其單獨立章；其主題（紅外成像、斷層掃描）已於[第 8 章](08-wavelengths-color-hyperspectral.md)（波長與高光譜）與[第 9 章](09-computational-imaging-survey.md)（計算成像綜覽）的外緣涵蓋。這也是本書章號從第 6 章跳至第 8 章的原因。
- **Lecture 12 為期末專題發表，無授課內容**，故亦不立專章。
- 課程中段的客座講次（如光場、Retrographic Sensing、BiDi Screen）依主題歸併至相關章節，詳見參考資料頁的逐講對照表。

## 體例說明

- **語言**：全書以繁體中文撰寫；專業術語於首次出現時附上英文原名。
- **每章結構**：各章開頭均有「對應講次／影片主題／對應講義」的 metadata 區塊，正文則統一採「導讀 → 核心內容 → 原理與系統 → 常見誤解 →（後續發展）→ 小結 → 延伸連結」的骨架，方便對照閱讀。
- **[術語表](glossary.md)**：彙整全書關鍵名詞的簡明定義，可隨時查閱；章節內首見的核心術語會連回此頁。
- **[參考資料](references.md)**：本書的權威來源頁，所有數字與人名的出處皆以此為準。
- **忠於 2009 年的歷史敘述**：本書內容忠實反映 2009 年當時的技術狀態；對於當年提出的預測（如 Lytro 前身、未來相機願景），會以「後續發展」小註補上截至今日的進展，而非改寫原始敘述。
