## 本講主問題

傳統相機在成像時，會將來自同一物點不同角度的光線積分為單一像素強度，導致光線角度資訊的喪失，無法還原 3D 空間中完整的光線狀態。我們該如何捕捉包含空間與角度的高維度光場（Light Fields）資訊？並且有了光場資訊後，我們能實現哪些傳統相機無法做到的應用（如事後對焦、像差校正）？

## 核心概念

| 概念 | 說明 | 書稿處理方式 |
|---|---|---|
| Light Field (光場) | 描述空間中所有位置與所有方向光線的函數。實際三維空間中的光場為 4D（例如兩個平面的參數化 $u,v$ 與 $s,t$）。在 2D 平面 (Flatland) 中則是 2D 參數 $x, \theta$。 | 納入導言，定義何謂光場以及 4D 參數化。 |
| Shift and Add (平移與相加) | 光場事後對焦的核心演算法。透過將不同視角的影像進行不同程度的平移（Shear）再相加，即可合成出聚焦於不同深度的影像。 | 納入原理與系統，說明事後對焦實作方法。 |
| Epipolar Plane Image (EPI) | 在 $x-\theta$ 空間中呈現光線的方法。無限遠的點呈現水平線，近處的點呈現傾斜線，點光源向外發散的光線呈現垂直線。 | 納入光場幾何視覺化說明。 |
| Space-Angle Trade-off | 在單一感測器上捕捉光場時，必須犧牲空間解析度（Spatial Resolution）來換取角度解析度（Angular Resolution）。 | 納入解析度取捨的討論。 |

## 重要細節

- 鏡頭的作用：鏡頭的作用是將外部世界的光場 (4D hologram) 複製一份到相機內部。傳統感測器只是將這份 4D 資訊壓縮成 2D 影像。
- 左/右順序反轉：在焦平面之前的失焦光束與在焦平面之後的光束，其空間的左/右顏色排列順序會發生反轉。這使得單憑模糊影像無法輕易還原出完整的高頻細節。
- 相位測量問題：波前測量存在 Phase wrapping (Modulo $2\pi$) 的問題。厚透鏡可被截斷成薄薄的菲涅耳透鏡 (Fresnel lens)，便是利用此原理。
- 光場相機架構：包括 Pinhole array (1908 年 Lippmann 提出)、Lenslet array (微透鏡陣列)、以及 Mask-based (光罩式)。
- 無透鏡陣列 (No Main Lens)：若移除主鏡頭僅剩微透鏡陣列，則能對極近距離的物體獲得極高的角度解析度，類似顯微鏡或生物複眼的概念。

## 原理與系統

- 原理／系統名稱：光場相機 (Light Field Camera) 與微透鏡陣列 (Lenslet Array)
- 光學或計算基礎：透過在主透鏡與感光元件之間插入微透鏡陣列。每一個微透鏡覆蓋感測器上的多個像素（例如 $14 \times 14$），這些像素分別記錄了透過主透鏡不同子孔徑 (Sub-aperture) 進入的光線角度。
- 支持證據（實驗／demo／發表系統）：Marc Levoy 等人在史丹佛開發的手持光場相機原型（Lytro 前身），將 1600 萬畫素分配為 $292 \times 292$ 的微透鏡，獲得 14x14 的角度解析度。
- 適用範圍與限制：單張照片即可記錄光場並實現事後對焦與視角微調，代價是最終影像的空間解析度會隨角度解析度等比例縮減。
- 與相鄰主題的關聯：與 Adaptive Optics 的 Shack-Hartmann 感測器原理同構；與相機陣列 (Camera Array) 在概念上等效。

## 書稿章節草稿

### 導讀

什麼是「光場 (Light Field)」？我們習慣將相機捕捉到的畫面視為一張 2D 的像素陣列，但真實世界的光線遠比這豐富。空間中任意一點，都有來自四面八方不同角度的光線交會。傳統相機會將這些不同角度的光線，透過鏡頭聚焦、積分，最終濃縮成感測器上的一個像素。這個「壓縮」過程讓我們失去了光線的方向資訊。本章我們將探討如何利用相機陣列或在感光元件前加入微透鏡，來捕捉包含空間與角度的「4D 光場」，進而達成「先拍照，後對焦」的魔法。

### 核心內容

在真實的 3D 世界中，光線可以被表示為一個 4D 的函數（例如穿越兩個平行平面的 $u, v$ 與 $s, t$ 座標）。為了簡化討論，我們常使用 2D 的「Flatland」來思考：光線在 2D 平面中可以用空間位置 $x$ 與角度 $\theta$ 來參數化，形成 $x-\theta$ 空間（又稱 Epipolar Plane Image, EPI）。

在 $x-\theta$ 空間中，光線的幾何行為變得十分直觀：
- 無限遠處的光源：所有光線平行，在 $x-\theta$ 空間中是一條水平線。
- 非常靠近的光源：向所有角度發散，是一條垂直線。
- 一般距離的點：隨著光線傳播，不同角度的光線會到達不同的位置，在 $x-\theta$ 空間中呈現為一條傾斜線 (Shear)。

### 原理與證據

1. **光場相機 (Light Field Camera)**：
   史丹佛大學團隊開發的光場相機原型，透過在傳統感光元件與主鏡頭間插入微透鏡陣列 (Lenslet Array)。光線經過主鏡頭聚焦後，微透鏡會將不同角度進入的光線分離，投射到其下方的不同像素上。雖然犧牲了空間解析度，但換取了角度資訊，使得感測器不僅記錄了「光有多亮」，更記錄了「光從哪裡來」。

2. **事後對焦 (Shift and Add)**：
   擁有光場資料後，對焦不再是物理鏡頭的專利，而是純數學的運算。我們可以將擷取到的不同視角（角度）的子影像，依照特定的位移量進行平移並相加 (Shift and Add)。平移量決定了合成影像的對焦深度。我們甚至可以設定非平行的對焦平面（如傾斜移軸效果）。

3. **天文與醫學的波前感測 (Shack-Hartmann Sensor)**：
   微透鏡陣列不僅用於攝影，在天文望遠鏡中，為了對抗大氣擾動帶來的波前 (Wavefront) 扭曲，科學家會發射雷射製造「導引星」，並透過 Shack-Hartmann 感測器（本質上也是一種光場感測器）測量光點在微透鏡下偏移的程度，藉此計算出波前的形變，並即時控制形變反射鏡 (Deformable Mirror) 進行補償。同樣的技術也被用於測量人眼水晶體的像差。

### 常見誤解

- **光場相機解析度太低？** 因為感測器總像素固定，這是一個空間解析度與角度解析度互換的零和遊戲 (Trade-off)。要獲得高品質的光場照片，需要極高畫素的底層感測器。
- **事後對焦只是模糊與銳利的濾鏡？** 不是。傳統軟體模糊無法無中生有還原被遮蔽的細節，光場的事後對焦是基於真實的光線幾何進行重建，能真正實現「看穿（部分）遮蔽物」的效果。

### 小結

光場攝影打破了自針孔相機以來的成像典範。透過理解光線在空間與角度中的 4D 分布，我們發現鏡頭其實只是完美地將外部的 4D 光場「複製」到相機內部。只要我們有合適的硬體與運算方法來攔截、記錄這個 4D 結構，相機將不再只是二維影像的記錄器，而是能夠捕捉並重建現實光學環境的強大工具。
