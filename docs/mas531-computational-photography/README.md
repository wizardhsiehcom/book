# MIT MAS.531 計算相機與攝影

本書是 MIT Media Lab 課程 **MAS.531 _Computational Camera and Photography_（Fall 2009，講者 Ramesh Raskar）** 的繁體中文精讀筆記，依課程逐字稿與講義整理而成。

計算攝影的核心理念是：相機不再只是被動記錄光線的暗箱，而是**光學硬體與演算法的共同設計（co-design）**。透過在照明、光圈、快門、感測器上加入「編碼」，再以軟體「解碼」，我們能捕捉傳統 2D 相機無法取得的高維度資訊——4D 光場、深度、光譜、偏振，甚至看穿遮蔽物與轉角。
